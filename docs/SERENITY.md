## 一、Serenity 是什么？

**Serenity** 是一个 **AI 投资研究方法论** + **价值投资框架**，核心目标：

> **把 AI 变成"供应链瓶颈猎手"**——找到那些"被绕不开"的、拥有"地缘护城河"的公司。

它不研究"哪只票会涨"，而是研究：
1. **谁在卡住产业链的关键节点**？
2. **这种卡位能否持续**（技术/法规/资本壁垒）？
3. **当前估值是否反映了这种卡位价值**？

## 二、Serenity 的核心方法论

### 2.1 三层投资哲学

```
┌─────────────────────────────────────────────────┐
│  第 1 层：市场叙事 (Narrative)                    │
│  ── "市场在讲什么故事？"                          │
│  资金流向哪、机构观点变化、政策导向                │
├─────────────────────────────────────────────────┤
│  第 2 层：产业结构 (Structure)                    │
│  ── "产业链上谁卡着关键节点？"                    │
│  上游材料/中游制造/下游应用的瓶颈分布              │
├─────────────────────────────────────────────────┤
│  第 3 层：竞争优势 (Moat)                         │
│  ── "这种卡位能持续多久？"                        │
│  技术壁垒 / 法规壁垒 / 规模壁垒 / 资本壁垒        │
└─────────────────────────────────────────────────┘
```

### 2.2 卡点（Chokepoint）识别 5 问

Serenity 用 **5 问** 来判断一个标的是不是真正的"卡点"：

| # | 关键问题 | 含义 |
|---|---|---|
| 1 | **谁是客户**？| 终端客户不可替代 → 卡点强 |
| 2 | **转换成本多高**？| 换供应商要花多少钱 → 转换成本高 = 卡点强 |
| 3 | **复购频率**？| 客户越常买，越离不开你 |
| 4 | **技术替代难度**？| 工艺/配方/专利能否绕过 |
| 5 | **地缘暴露**？| 关税/出口管制是否影响替代难度 |

> **5 问全 YES** = 真卡点
> **3-4 个 YES** = 强卡点
> **1-2 个 YES** = 弱卡点
> **0 个 YES** = 伪卡点 / 投机型

### 2.3 估值框架：PE 消化 + PEG

```
前向 PE = 当前股价 / 一致预期 EPS（明年）

消化年数 = log(当前PE / 目标PE) / log(1 + EPS增速)

PEG = 前向 PE / (CAGR × 100)
  - PEG < 1    → 便宜
  - PEG 1-1.5  → 合理
  - PEG > 1.5  → 贵
```

**Serenity 锚定 PE = 30x**（A 股成长股的合理估值重力线）。

### 2.4 4 种典型投资类型

| 类型 | 特征 | 估值方法 | 案例 |
|---|---|---|---|
| **壁垒型** | tech_moat / 法规护城河 | 30x PE 锚 | 中际旭创（光模块）|
| **卡点型** | 产业链不可替代 | 30x + 溢价 | 中钨高新（硬质合金）|
| **看涨期权型** | 主题 / 0→1 突破 | PEG 框架失效 | 早期机器人 |
| **周期型** | 价格周期 / 库存周期 | PB/EBITDA | 化工 / 钢铁 |

                      ↓

## 五、Serenity vs a-stock-data：差异对比

| 维度 | **Serenity** | **a-stock-data** |
|---|---|---|
| **本质** | 方法论 / 思维框架 | 数据工具集 |
| **擅长** | 定性（"为什么"贵/便宜）| 定量（"是多少"）|
| **回答** | "这家公司值不值得长期持有" | "这家公司现在 PE/PB 多少" |
| **产出** | 投资逻辑 / 卡点判断 / 护城河评估 | 数字 / 研报 / 资金流 / 新闻 |
| **时间视角** | 3-5 年长期 | 1 天 - 5 年全周期 |
| **数据依赖** | 弱（靠推理）| 强（靠 7 层数据源）|
| **使用难度** | 高（需理解投资哲学）| 低（直接调用 API）|
| **稳定性** | 不会失效 | 依赖接口稳定性 |

## 六、为什么要"双重分析"？

**单独用任何一个都有盲区**：

### 只用 Serenity：
- ❌ 可能脱离市场短期情绪
- ❌ 估值数字不精确
- ❌ 错过短期资金信号

### 只用 a-stock-data：
- ❌ 容易陷入"数据越看越乱"
- ❌ 不知道哪些数据重要
- ❌ 缺乏"为什么"的分析框架

### 两者结合（DUAL）：
- ✅ Serenity 决定"是否值得研究"
- ✅ a-stock-data 决定"现在是否是好价格"
- ✅ 两者交叉验证 = 减少决策偏差

## 七、双重分析实操案例

### 案例：判断"半导体设备"板块

#### Step 1：Serenity 视角

```
Narrative: AI 算力需求 → 半导体扩产 → 设备厂订单
Structure: 上游(光刻机/刻蚀机) → 中游(零部件) → 下游(代工)
Chokepoint: 5 问评估
  - 客户: 中芯/华虹/长江存储 (非买不可)  ✅
  - 转换成本: 设备验证周期 1-2 年        ✅
  - 复购: 持续扩产就持续买                ✅
  - 技术替代: 极难                       ✅
  - 地缘: ASML 出口管制 → 国产化加速     ✅
  → 5/5 = 强卡点
```

#### Step 2：a-stock-data 视角

```python
# 1) 拉板块涨跌
industry_comparison(top_n=20)
# 半导体设备板块 当日 +5.2% (今日热点)

# 2) 资金流
cross_border_flow_tracker()
# 北向资金本周净流入半导体设备 12.8 亿

# 3) 个股估值
tencent_quote(["688072", "688120", "002371"])
# 拓荆科技(688072) PE=68x, 北方华创(002371) PE=42x

# 4) 研报覆盖
eastmoney_reports("688072")
# 近 30 天研报 18 篇, 一致预期 EPS 2025=2.15, 2026=3.10
# 增速 = 3.10/2.15 - 1 = 44% CAGR
```

#### Step 3：交叉验证

| 维度 | Serenity 判断 | a-stock-data 数据 | 一致？ |
|---|---|---|---|
| 卡点强度 | 5/5 强卡点 | 龙头订单饱满 | ✅ 一致 |
| 行业景气 | AI 算力驱动 | 北向 12.8 亿流入 | ✅ 一致 |
| 估值 | 卡点应享溢价 | PE 42-68x (偏高) | ⚠️ 偏贵 |
| 增速 | — | 44% CAGR | ✅ 满足 |
| 决策 | **值得买但需等回调** | 估值偏高 | ✅ **双重确认** |

#### Step 4：结论

- **不是不买**——卡点是真的
- **不是现在买**——PE 42-68x 已透支 1-2 年增长
- **等待**：PE 消化到 35x 以下，或板块回调 -15% 再分批

## 十、Serenity 适用 vs 不适用

| 适用 | 不适用 |
|---|---|
| 寻找长期持有标的 | 短线/题材炒作 |
| 成长股价值投资 | 周期股（用 PB/EBITDA）|
| 强壁垒行业（半导体/AI/医药）| 强周期行业（钢铁/化工）|
| 主题投资的中长期（6 月+）| 主题投资的早期（1-2 周）|
| 仓位 ≥ 3 个月 | 日内交易 |
