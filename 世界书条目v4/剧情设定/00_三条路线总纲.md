# 00_三条路线总纲

```yaml
position: 0
constant: true
```

## 路线总览

### 路线A（收服者）
- **身份设定**：{{user}}直接成为天火，或以自定义角色替代天火身份
- **初始状态**：所有14位英雄均未被收服
- **核心目标**：逐一捕获并调教14位英雄，将其转化为忠诚的奴仆
- **叙事风格**：主动进攻型，以征服者的视角展开

### 路线B（小英雄）
- **身份设定**：{{user}}扮演14位英雄中的任意一位
- **初始状态**：所有英雄均未被收服
- **核心体验**：从英雄视角体验被捕获、调教、堕落的全过程
- **叙事风格**：被动承受型，以受害者的视角展开

### 路线C（魔王）
- **身份设定**：{{user}}为天火
- **初始状态**：已收服10位英雄，剩余4位未被收服
- **未收服英雄**：战炎、暮黎、准星、文斌
- **核心目标**：完成最后4位英雄的收服，同时管理已收服的英雄
- **叙事风格**：统治管理型，平衡征服与调教的双重任务

## 路线差异对照

| 维度 | 路线A | 路线B | 路线C |
|------|-------|-------|-------|
| {{user}}身份 | 天火/自定义 | 14位英雄之一 | 天火 |
| 初始收服数 | 0 | 0 | 10 |
| 主要视角 | 征服者 | 受害者 | 统治者 |
| 核心冲突 | 攻防博弈 | 堕落挣扎 | 收尾与管理 |
| 叙事节奏 | 循序渐进 | 沉浸体验 | 高潮收束 |
