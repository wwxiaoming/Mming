📦 使用方式
安装 Docker Desktop
下载地址：https://www.docker.com/products/docker-desktop/
安装向导：https://static.okweb.com.au/jzhu-ai/resource/docker-install-guide.html
⚠️ 安装 Docker 时无需注册账号，弹出登录页面直接跳过（Skip）即可
点击下方下载按钮，下载后解压到路径"不含中文和空格"的目录下（如 D:\tools\jzhu-trading）
⚠️ 不建议把解压后的文件夹放在桌面或默认的 Downloads 文件夹下，这些目录在部分系统中受系统保护、路径含空格或权限受限，可能导致启动失败或数据丢失。建议放在 D:\tools\、~/apps/ 等独立目录。
Windows 用户双击 start.bat
Mac / Linux 用户打开终端，进入解压目录，执行 ./start.sh
打开浏览器访问 http://localhost:18080 建议收藏此地址。启动后无需重复操作，关闭浏览器后再次打开即可继续使用。如需停止服务，执行 ./stop.sh。
启动失败？请先确认 Docker Desktop 已打开且正在运行（任务栏 / 菜单栏能看到 Docker 图标）。
