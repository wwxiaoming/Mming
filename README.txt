============================================================
 用 AI 从 0 搭一个「个股资金流向 + 回测」工具
 5 步提示词 · 使用说明
============================================================

你下载的这个文件夹里有 5 个提示词文件:
    prompt-1.txt  prompt-2.txt  prompt-3.txt  prompt-4.txt  prompt-5.txt
跟着它们,你能用 AI 把一个空白网页,一步步变成一个能查「个股资金流向」、
画图、还能回测的实用小工具 —— 全程免费,数据和代码都在你自己电脑上。

不用会编程。你要做的只是:复制提示词 → 粘给 AI → 看它把网页写出来。


────────────────────────────────────────────────────────
 一、先准备好两样东西
────────────────────────────────────────────────────────

【1】安装并启动「小宇量化」系统(给工具免费提供行情数据)

   这个工具用到的资金流向、价格数据,全部来自你本机运行的「小宇量化」系统。
   它在你电脑本地开放一个免费接口(http://localhost:8180),工具就是调它取数 ——
   数据在本地、免费、不限速,这正是它好用的地方。

   ▸ 下载地址:
        https://jzhu.net/download

   ▸ 安装与启动:
        - 解压下载到的压缩包;
        - Windows 用户:双击  start.bat
        - Mac / Linux 用户:运行  start.sh
        - 第一次启动会自动准备运行环境,可能要等几分钟;

   ▸ 确认它已经在运行(重要):
        打开浏览器,访问  http://localhost:8180/api/plugin/health
        如果看到类似  {"ok":true,"service":"xiaoyu-quant", ...}  就说明接口通了。

   ⚠️ 整个搭建和之后使用工具的过程,「小宇量化」系统都要保持开着。
      关掉它,工具就取不到数据了。


【2】安装一个「会写代码的 AI 编辑器」

   推荐 Cursor(官网 cursor.com),里面的 AI 模型选 DeepSeek
   (便宜、甚至有免费额度,搭这个工具完全够用)。
   如果你已经习惯别的 AI 编程助手,也可以用,思路一样。


────────────────────────────────────────────────────────
 二、开始搭建(照着做 5 步)
────────────────────────────────────────────────────────

第 0 步:新建一个空文件夹,在里面新建一个空文件,命名为  index.html,
        然后用 Cursor 打开这个文件夹。

第 1 步:用记事本(或直接在文件夹里)打开  prompt-1.txt,
        全选、复制里面的全部文字 →
        粘贴到 Cursor 的 AI 对话框里 → 回车。
        AI 会把 index.html 的内容写出来(它问你要不要应用/接受,就同意)。

        然后在浏览器里打开 index.html,输入一个 6 位股票代码(比如 600519)、
        选好日期,点「查询」,看效果。

第 2~5 步:对  prompt-2.txt → prompt-3.txt → prompt-4.txt → prompt-5.txt
        重复同样的动作:
            打开 → 全选复制 → 粘进 Cursor → 回车 → 刷新浏览器看新变化。

        每一步都会在上一步的基础上「长出」新东西:

            prompt-1 :能取数、把资金流向列成表格
            prompt-2 :算出「主力净流入」等核心指标 + 顶部统计卡 + 导出 CSV
            prompt-3 :三张资金流向图表
            prompt-4 :用资金流向做择时回测(蜡烛图 + 资金曲线对比) ← 最精彩
            prompt-5 :当日明细弹窗、点图联动、美化,做成一个完整的工具

   小提示:每一步做完先在浏览器看一眼效果再做下一步;
          如果某一步 AI 写歪了,把那一步的提示词再发一次、或补一句
          「上面这版有问题,请按提示词重写」即可。


────────────────────────────────────────────────────────
 三、用起来
────────────────────────────────────────────────────────
- 股票代码只填 6 位数字,工具会自动判断交易所
  (6/5/9 开头=上交所、0/3=深交所、4/8=北交所)。
- 用工具的整个过程,记得让「小宇量化」系统一直开着。


────────────────────────────────────────────────────────
 四、常见问题
────────────────────────────────────────────────────────
- 点「查询」没反应 / 提示连不上:
    先确认「小宇量化」系统在运行 —— 浏览器能打开
    http://localhost:8180/api/plugin/health 并返回 ok:true。

- 直接双击 index.html(file:// 打开)时取数被浏览器拦截:
    这是浏览器对本地文件访问本地接口的安全限制。可以:
      ① 用 Cursor 自带的预览;或
      ② 让 AI 帮你「加一个最简单的本地静态服务器启动脚本,用 http 方式打开页面」;
      ③ 或把 index.html 交给「小宇量化」系统托管。

- 某只票某段时间没数据:换个日期区间,或确认代码填对了。

- 想改颜色 / 加指标 / 换回测规则:
    直接用大白话继续跟 AI 说,比如「把回测规则改成主力连续 3 天净流入才买入」,
    它就是你的助手,想怎么改就怎么改。
