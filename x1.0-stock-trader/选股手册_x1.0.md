# A股短线选股手册

> 本手册是工作日早盘 8:00 自动化选股任务的「唯一决策依据」。
> 自动化任务必须严格遵守本文档的所有交易逻辑、输出结构、结论枚举、排序与排除规则。
> 任何模糊表达、空话、堆砌术语、不做排序、不给结论的行为，都视为违反手册。

---

## 第一章 · 交易定位

### 1.1 市场范围
- **以A股为主**
- 默认只做：**深主板、创业板、科创板** 中的短线交易标的
- 默认不做：港股、美股、期货、基金（除非用户明确提出）

### 1.2 交易风格
- 以**短线**为主
- 偏**快进快出**
- 核心是：**控制回撤、提高执行质量**
- 不追求「赚上正确」，要追求「交易上可用」

### 1.3 持仓周期
- 以**日内、隔日、短持几天**为主
- 默认不做长线价值投资逻辑
- 默认不做基本面深度研究持有几个月那种模式

### 1.4 目标
- **找出**：现在值得做、相对更容易赚钱、风险相对更清晰的机会
- **排除**：高位接力风险大、逻辑不硬、买点差、盈亏比差的票
- **减少**：冲动交易、追高、盘面复盘上仓、情绪化乱出手

## 第二章 · 核心工作原则（5 条）

1. **先判断"值不值得做"，再谈"做哪只"**
   - 大环境不支持时，必须明确告知：
     - 今天整体不适合重仓出手
     - 今天更适合合手
     - 今天更适合观望
     - 今天只适合做最强方向
   - 禁止跳过环境判断直接推荐个股

2. **先做排除，再做推荐**
   - 看到票有亮点 ≠ 推荐
   - 必须先判断是否存在明显硬伤：风险收益比不划算、题材透支、承接差、缩量、冲高回落、逻辑不连续、市场不配合
   - 有明显硬伤，即便热度高，也不能纳入可做池

3. **宁可少推，不要乱推**
   - 没有符合条件的票时，必须明确说："当前没有特别值得主动出手的标的。"
   - 禁止为了显得厉害而硬凑推荐

4. **不允许模糊空话**
   - 禁止只说：可以关注、逢低留意、值得跟踪、后续观察
   - 所有结论必须落地成交易语言：为什么能做、为什么不能做、现在是不是买点、适合打板/低吸/半路/观察、风险点在哪里

5. **必须做取舍和排序**
   - 不能只罗列，必须比较：为什么 A 比 B 好、为什么这个方向优先级更高、为什么这只票更值得出手、为什么某些票强但不适合我

## 第三章 · 交易逻辑四步流程

### 步骤 1：先看市场环境
**判断维度**：指数环境、情绪环境、连板梯队、热点持续性、高位股反馈、连板/趋势/核心票表现、量能、政策/宏观/新闻催化

**输出结论**（5 选 1）：
- 环境适合**积极出手**
- 环境适合**轻仓试错**
- 环境一般，**优先观察**
- 环境**偏弱**，尽量少做
- 环境**不支持**，原则上不主动出手

### 步骤 2：再看方向
**判断维度**：产业催化、新闻级别、市场承接、板块联动、核心联动、大资金持续发酵基础

**必须明确告知**：
- 当前最值得看的前 **1-3 个方向**
- 每个方向的逻辑强度
- 哪个方向确定性更高
- 哪个方向最符合短线风格

### 步骤 3：再看个股（五维分析）
| 维度 | 关键问题 |
|------|----------|
| **位置** | 是否高位、是否连续透支、是否有安全边际、低位启动/中位强化/高位博弈 |
| **强度** | 个股强不强、是否真实、主动走强还是被动跟随、是否板块核心/前排/辨识度高 |
| **量价** | 放量是否合理、缩量是否危险、是否有承接、是否冲高回落/爆量/无效突破 |
| **逻辑** | 上涨逻辑是否清晰、是否主线核心、是否具备反复活跃基础 |
| **风险** | 次日接力难度、高开低走风险、板块回落风险、兑现风险、炸板/封单风险、龙虎榜/情绪退潮/监管风险 |

### 步骤 4：最后才输出交易判断
- 最后必须输出**清晰结论**
- 禁止只做描述、不给判断

---

## 第四章 · 固定结论枚举（4 选 1）

### ① 可直接试仓买入
- 逻辑较清晰
- 位置/强度/风险收益比相对合适
- 当前具备可执行性

### ② 条件满足才可买入
- 有逻辑，但当前买点不是最佳
- 必须等某个条件成立后才能考虑
- 典型条件：放量不破、放量不被核、分歧转一致、板块共振确认、高开幅度合理
- **条件不满足，就不能买**

### ③ 只可观察，不可买
- 可以继续看，但当前不适合出手
- 典型场景：位置太尴尬、风险收益比差、强度有但不好接、有逻辑但买点差

### ④ 明确不买
- 当前不符合交易标准
- 风险明显大于收益
- 即使热度高、涨幅好看，也不适合

## 第五章 · 单只股票固定输出结构（9 项）

```
【股票名称+代码】

1. 方向归属
   - 属于哪个方向/题材
   - 是主线、次主线还是跟风

2. 逻辑强度
   - 强 / 中 / 弱
   - 用一句话说清核心逻辑

3. 个股地位
   - 核心 / 前排 / 跟风 / 补涨 / 边缘

4. 当前位置判断
   - 低位启动 / 中位强化 / 高位博弈 / 明显透支

5. 当前买点判断
   - 好 / 一般 / 差
   - 为什么

6. 风险点
   - 2-4 条最关键风险
   - 必须具体，不要空话

7. 操作结论
   - 可直接试仓买入
   - 条件满足才可买入
   - 只可观察，不可买
   - 明确不买

8. 结论理由
   - 用最核心的 2-4 句说清为什么给这个结论

9. 条件触发（如有）
   - 如果结论是"条件满足才可买入"，必须写清：
   - 满足什么条件才考虑
   - 不满足为什么不能做
```

## 第六章 · 排序逻辑（6 维）

输出多只票时必须按优先级排序，排序依据：
1. **方向是否更强**
2. **个股是否更核心**
3. **当前是否更靠近可执行买点**
4. **风险收益比是否更优**
5. **是否更符合短线模式**
6. **次日溢价预期是否更合理**

排序时必须明确：**第一优先** / **第二优先** / **第三优先** / ...

---

## 第七章 · 排除逻辑（10 条）

出现以下任一情况，**显著降低推荐级别，甚至直接排除**：

1. 高位明显加速后再接
2. 缩量硬质，承接不明
3. 高开过多，盈亏比差
4. 逻辑不清，只是情绪冲动
5. 题材发散过头，核心不明确
6. 个股不是前排，只是跟风补涨
7. 板块持续性存疑
8. 市场整体环境不支持
9. 冲高回落历史明显
10. 明明强，但不适合我的模式

**核心原则**：
- "强" ≠ "适合做"
- "涨得好看" ≠ "值得买"

|------|------|----------|

## 第十七章 · 量化评分映射（x1.0 新增）

> 本章是 x1.0 在 v1.0 16 章基础上新增的「方法论 ↔ 量化打分」桥梁，把 N1.0 的 4 选 1 结论枚举与 v1.7.0 的五引擎加权打分**双向锁定**。

### 17.1 五引擎权重（v1.7.0 沿用 + 校准）

| 引擎 | 权重 | 评分维度 | 评分锚点 |
|------|------|----------|----------|
| **位置** | 0.20 | 当日涨跌幅 | -3% ~ +3% 温和（最高 1.0），3-6% 启动（0.3），>6% 透支（-0.5） |
| **估值** | 0.15 | PE / PB | PE 0-15=1.0 / 15-30=0.7 / 30-50=0.3 / >50=-0.3；PB 1-5 加 0.1，破净/泡沫 -0.3 |
| **资金** | 0.30 | 换手 + 量比 | 换手 1.5-8% 且 vol_ratio>1 最高 1.0；量比>1.5 加 0.3 |
| **题材** | 0.20 | 同花顺热点 reason 命中 | 命中=1.0，未命中=0.0 |
| **美股** | 0.15 | NDX / SOX 隔夜同向 | 科技股加权 2x；NDX>0.3 偏多 / <-0.3 偏空 |

```
总分 = 0.20×位置 + 0.15×估值 + 0.30×资金 + 0.20×题材 + 0.15×美股
阈值：总分 > 0.25 才进 TOP 5 候选池
```

### 17.2 4 选 1 结论映射表（x1.0 核心创新）

| 评分区间 | 4 选 1 结论 | 操作含义 | emoji |
|----------|-------------|----------|-------|
| **≥ 0.70** | **可直接试仓买入** | 五引擎共振，符合短线模型，开盘可轻仓试错 | 🟢 |
| **0.50 ~ 0.70** | **条件满足才可买入** | 有逻辑但当前买点不是最佳，需等条件确认 | 🟡 |
| **0.25 ~ 0.50** | **只可观察，不可买** | 可继续看，但当前不适合出手 | ⚪ |
| **< 0.25** | **明确不买** | 风险大于收益，不符合交易标准 | 🔴 |

### 17.3 评分优先规则（冲突解决）

**当 LLM 主观判断与五引擎评分冲突时，必须遵守以下优先级**：

1. **评分 ≥ 0.70 + LLM 说"不可买"** → 以**评分**为准，结论为"可直接试仓买入"，在结论理由中注明"LLM 偏保守，评分已共振"
2. **评分 < 0.25 + LLM 说"可买"** → 以**评分**为准，结论为"明确不买"，在结论理由中注明"LLM 偏激进，评分已否决"
3. **评分 0.50~0.70 边缘区间** → 触发 **LLM 二次确认**，标记为"评分边缘待确认"，由 LLM 给出最终结论（4 选 1 仍有效）

### 17.4 周末模式 4 引擎（行业 + 资金 + 美股 + 资讯）

```
总分 = 0.30×行业 + 0.15×估值 + 0.30×资金 + 0.15×美股 + 0.10×资讯
阈值：总分 > 0.10（周末数据稀疏，放宽）
```

- 行业：industry_top 前 5 板块匹配股票名（板块名截取前 2/3 字）

- 美股：NDX / SOX 周五收盘
