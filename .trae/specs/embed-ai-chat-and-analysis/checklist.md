# Checklist

- [ ] 全局抽屉入口可见，点击可从右侧滑出 AI 助手
- [ ] 抽屉状态（开/关）在刷新页面后保持
- [ ] 首次打开显示欢迎语 + 至少 4 个预置快捷指令气泡
- [ ] 点击快捷指令气泡等同于自动发送对应文本
- [ ] 输入框输入文字后按 Enter 能正常发送，按 Shift+Enter 换行
- [ ] 助手回复以流式方式逐字渲染，期间可点"停止生成"中断
- [ ] 消息列表支持 Markdown 与代码块渲染
- [ ] 多轮对话中后文能引用前文（验证"AI 会记住上下文"）
- [ ] 输入框实时显示 `当前字数 / 上限`，超限禁止发送
- [ ] 设置面板包含 Base URL、API Key、模型名、Temperature、最大上下文轮数 N、超时、自定义系统提示词、请求模式
- [ ] "测试连接"按钮可调通 DeepSeek / OpenAI / Ollama 等 OpenAI 兼容服务并显示模型数
- [ ] 配置保存后无需刷新，抽屉模型下拉立即同步
- [ ] API Key 输入框支持"显示/隐藏"
- [ ] 未配置 API 就发送消息时，抽屉自动弹出设置面板高亮必填项
- [ ] 配置与历史均写入 localStorage，刷新后保留
- [ ] 提示用户"API Key 仅保存在本机浏览器"
- [ ] 业务页面（行情页/策略页）可点击按钮把当前上下文（K 线/策略代码）注入本轮对话
- [ ] 附加上下文不进入历史，仅作用于本轮
- [ ] 用户可自定义快捷指令（增/删/改/排序），支持 `{{占位符}}` 模板
- [ ] 每条助手消息末尾始终显示免责声明
- [ ] 浏览器直连失败时可切换为"经本站代理"模式（CORS 兜底）
- [ ] 请求支持超时与 AbortController
- [ ] 整体 UI 在桌面与中等宽度屏幕下不遮挡关键操作
