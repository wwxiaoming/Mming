# 道具图鉴 + 五大区设定 格式精简化 Spec（参照精能大陆）

## Why
当前 `01_人类安全区.md` / `02_魔神会控制区.md` / `07_道具图鉴.md` 三份设定含大量通用杂项段（概述/关键事件/关键势力/物品概述/外观/核心功能/使用机制等），与精能大陆"分类+条目"紧凑写法不一致。参照 `/workspace/精能大陆(小光).txt` 中"魔物与陷阱分类"+"星小光状态系统"+"五大分区·特色与威胁锚点"的紧凑写法，删除 3 份文件中的通用杂项段，仅保留具体条目（5 大区/4 大类道具），使 LLM 在单次激活中能更精准命中条目。

**用户反馈约束**：
- **删除**：物品概述/外观/核心功能/使用机制（道具图鉴）— 用户明确"什么物品概述外观三条路线等都删掉"
- **删除**：概述/关键事件/关键势力（地区设定）— 通用杂项段同样需要清理
- **保留**：4 大类道具条目 + 5 大区城市条目（精能大陆格式：每区 4 维度 = 方位 + 风貌 + 核心设施 + 标志威胁）
- **参照精能大陆格式**：紧凑条目化，标题简洁
- **保持不动**：YAML 字段（key/secondary_keys/enabled/position/order），仅在 `content: |` 块内重写
