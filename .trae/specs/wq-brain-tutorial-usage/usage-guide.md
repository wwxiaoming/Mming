## 一、教程站结构概览

教程站首页（[guide/](https://alphadoc.biglongxia.com/guide/)）是整站的总入口，包含以下板块：

| 板块 | 说明 |
|------|------|
| 📖 阅读须知 | 教程使用约定，含 ⚠️ 警告和 💡 提示的标记体系 |
| 📚 教材目录 | 11 章内容的索引表，每章可点击跳转 |
| 🗺️ WQ学习计划 | 7 天分阶段学习计划，是实操的核心路线图 |
| 📌 遇到问题 | 4 级求助路径（查阅章节 → 群内 → 社群交流 → 联系团队） |

### 标记符号含义

教程中使用了两种关键标记：

- **⚠️ 警告**：容易踩的坑，忽视可能导致账号封禁、资格丧失等严重后果，**必须逐条注意**
- **💡 提示**：实用技巧，可帮助提高效率

### 推荐阅读策略

- **首次使用**：按顺序从头到尾阅读第 01-11 章，不要跳章
- **遇到具体问题**：直接跳转到对应章节查阅，每章独立成文
- **日常执行**：以「🗺️ WQ学习计划」为主线，遇到不懂的概念到对应章节或 ima 知识库查询

## 三、7 天学习计划详细解读

教程站的「🗺️ WQ学习计划」是实操核心，分为两个阶段：**第一阶段（普通用户 → 正式顾问）** 和 **第二阶段（进阶赚钱）**。以下是第一阶段的逐日解读。

### 第 1 天：环境搭建与概念入门

**核心目标**：注册账号 + 安装工具 + 让工具跑起来 + 理解基本概念

| 任务 | 说明 |
|------|------|
| 学习教程 01-04 章 | 快速过一遍项目背景、FAQ、云电脑申请、注册流程 |
| 完成注册与工具安装 | 按 03-05 章指引，申请云电脑 → 注册 BRAIN 账号 → 安装 LQS Alpha 工具 → 启动运行 |
| 加入 WQ 知识库 | 收藏 ima 知识库，后续概念查询的主要渠道 |
| 概念学习 | 向知识库提问：什么是 Alpha、什么是量化、顾问薪酬结构、Genius Program 等级与季度奖金 |

**注意**：第一天搞不清状况是正常的，"几乎所有顾问都是这样过来的"。

### 第 2 天：日常提交 + 概念深化

**核心目标**：开始提交 Alpha + 建立每日检查习惯

| 任务 | 说明 |
|------|------|
| 中午 12 点前查看工具 | 检查是否有可提交的 Alpha（绿色结果），有则提交 |
| 下午 3 点后查积分 | 在 BRAIN 官网查看积分变化 |
| 每日查邮件 | 注册邮箱会收到奖牌通知、课程预约、新人引导等官方邮件 |
| 听播客 + 提问 | 收听《Alpha模型赚钱逻辑》播客，不懂的概念问 ima 知识库 |
| 概念学习 | 向知识库提问：Long Count 与 Short Count 的定义及用法 |

### 第 3 天：理论学习深化

**核心目标**：持续提交 + 学习中性化概念

| 任务 | 说明 |
|------|------|
| 中午 12 点前提交 | 重复日常提交流程 |
| 官方课程 | 如收到课程预约邮件，可预约听课（没时间可看往期录播） |
| 播客学习 | 收听《构建工业级量化Alpha》播客 |
| 概念学习 | 向知识库提问：什么是中性化（Neutralization） |

### 第 4 天：自相关性学习

**核心目标**：持续提交 + 理解 Alpha 质量指标

| 任务 | 说明 |
|------|------|
| 日常提交流程 | 同前几日 |
| 概念学习 | 重新学习教程 06 章，向 ima 提问：什么是自相关性，如何降低自相关性 |
| 官方课后作业 | 如完成了直播课，可尝试做课后作业（不做不影响成为顾问） |

### 第 5 天：参数调优

**核心目标**：持续提交 + 学会调整策略

| 任务 | 说明 |
|------|------|
| 日常提交流程 | 同前几日 |
| 策略调整 | 如果连续两天没有跑出可提交 Alpha，按照教程 06 的官方推荐表更换数据集 + 中性化组合，从第一阶段重新跑 |
| 官方课程 | 持续参与官方直播课程（或看录播） |

### 第 6 天：关键卡点 — 问卷 + Workday

**核心目标**：达到 10,000 分金牌后，完成问卷和 Workday

这是整个流程中**最关键的一天**，直接决定能否成为顾问：

| 任务 | 说明 |
|------|------|
| 先提交满 20 个 Alpha | 拿到金牌后尽快提交满 20 个，之后以每 2 天 1 个的频率维持 |
| 在邮箱中找问卷入口 | 完成「基础测试问卷」——**两次机会，两次均错则永久失去资格** |
| 在邮箱中找 Workday 入口 | 按照教程 08 填写，**务必仔细检查，信息错误会使审核时间翻倍** |

> ⚠️ **严格执行顺序**：必须等问卷**通过后**再提交 Workday！如果问卷两次都没通过，而你同时提交了 Workday，则此人永久失去资格。

### 第 7 天及之后：等待顾问审批

**核心目标**：保持活跃 + 自学进阶 + 等待审批

| 任务 | 说明 |
|------|------|
| 维持提交频率 | 先提交满 20 个 Alpha，之后每 2 天 1 个 |
| 自学进阶 | 学习数据集、中性化搭配规则，探索 Alpha 参数含义 |
| 利用官方资源 | 官方论坛、Learn 页面、官方推荐公众号 |
| 等待审批 | 审核周期 2 周至 1 个月，保持活跃有助于提升排队顺序 |

> 达到 10,000 分后，官方会在一周内发邀请邮件；如一周后未收到，自行发邮件至 support@worldquantbrain.com 咨询。

## 四、关键警告与常见踩坑汇总

### 🔴 致命级（可能导致永久失去资格）

| 风险点 | 详情 |
|--------|------|
| **基础测试问卷两次失败** | 只有两次机会，第二次审核比第一次严苛。机审查重直接挂掉——**不要抄别人的答案** |
| **问卷未通过就提交 Workday** | 如果问卷两次都失败且 Workday 已提交，此人永久失去兼职资格 |
| **Workday 信息填写错误** | 错误会导致审核时间翻倍及以上。填写内容是签约合同的信息来源，必须真实准确 |

### 🟠 高风险级（可能导致封号）

| 风险点 | 详情 |
|--------|------|
| **在社群外提及工具或教程站** | **严禁**在官方公众号群等公开场合提到 Alpha 辅助工具或教程站，"低调干大事，闷声发大财" |
| **账号风控触发面试** | 达到 10,000 分后有极小概率触发账号冻结 + 面试。需准备身份证原件，回答面试官问题，联系导师获取支持 |

### 🟡 注意级（影响效率）

| 风险点 | 详情 |
|--------|------|
| **工具被 Windows Defender 标记** | 安装 LQS Alpha 工具需关闭防火墙和 Windows 安全中心（FAQ 第 18 条） |
| **使用云电脑而非自用电脑** | 教程要求使用云电脑固定 IP，避免 IP 变动导致账号风险 |
| **审核周期长** | Workday 审核 2 周至 1 个月属正常范围，耐心等待，期间多提交 Alpha 保持活跃 |
