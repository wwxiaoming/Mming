# 股票分析网站 Spec

## Why
为用户提供一个直观、专业的股票分析工具，支持股票搜索、价格走势图表展示、技术指标分析和关键财务数据查看，帮助用户快速了解股票行情并做出投资决策。

## What Changes
- 新建一个基于 Next.js + React + TypeScript 的股票分析单页应用
- 实现股票搜索与自动补全功能
- 实现交互式价格走势图表（K线图 / 折线图切换）
- 实现常用技术指标叠加（MA均线、MACD、RSI）
- 实现股票关键财务数据展示面板
- 支持多时间周期切换（日/周/月）
- 使用免费公开的股票数据 API 获取数据

## Impact
- Affected specs: 无（全新项目）
- Affected code: 全新项目，不涉及现有代码

## ADDED Requirements

### Requirement: 股票搜索功能
系统 SHALL 提供股票搜索输入框，用户输入股票代码或名称时实时展示匹配结果，用户可选择搜索结果中的股票进行查看。

#### Scenario: 用户通过股票代码搜索
- **WHEN** 用户在搜索框输入 "AAPL"
- **THEN** 系统显示匹配的股票建议列表，包含股票代码和名称

#### Scenario: 用户选择搜索结果
- **WHEN** 用户点击搜索结果中的某只股票
- **THEN** 系统加载并展示该股票的详细分析页面

### Requirement: 价格走势图表
系统 SHALL 以交互式图表展示股票历史价格数据，支持 K线图（蜡烛图）和折线图两种视图模式切换。

#### Scenario: 查看日K线图
- **WHEN** 用户选择"日K"时间周期和"K线图"模式
- **THEN** 系统展示该股票的日级别蜡烛图，包含开盘价、收盘价、最高价、最低价

#### Scenario: 切换到折线图
- **WHEN** 用户将图表模式切换为"折线图"
- **THEN** 系统以折线图展示收盘价走势

#### Scenario: 切换时间周期
- **WHEN** 用户选择"周K"或"月K"时间周期
- **THEN** 系统重新加载对应周期的数据并更新图表

### Requirement: 技术指标叠加
系统 SHALL 支持在价格图表上叠加常用技术指标，用户可独立开关各指标显示。

#### Scenario: 显示MA均线
- **WHEN** 用户开启 MA 均线指标
- **THEN** 图表上叠加显示 MA5、MA10、MA20、MA60 均线

#### Scenario: 显示MACD指标
- **WHEN** 用户开启 MACD 指标
- **THEN** 图表下方显示 MACD 指标子图，包含 DIF、DEA 和柱状图

#### Scenario: 显示RSI指标
- **WHEN** 用户开启 RSI 指标
- **THEN** 图表下方显示 RSI 指标子图，包含 RSI6、RSI14、RSI24 三条线

### Requirement: 财务数据面板
系统 SHALL 在股票详情页展示关键财务指标面板，包括市值、市盈率、每股收益、股息率等。

#### Scenario: 查看股票财务数据
- **WHEN** 用户选中某只股票
- **THEN** 页面侧边栏或下方展示该股票的关键财务数据

### Requirement: 响应式布局
系统 SHALL 适配桌面端和移动端设备，在不同屏幕尺寸下保持良好的可用性和可读性。

#### Scenario: 移动端访问
- **WHEN** 用户在手机浏览器中打开网站
- **THEN** 页面布局自动适配移动端，搜索框、图表和财务面板垂直排列
