# 帝王战队 五大区与道具图鉴 整体优化 Spec

## Why
当前 `01_人类安全区.md` 与 `02_魔神会控制区.md` 中的"五大区"虽然已对齐小说原文，但缺乏精能大陆式的差异化结构（风貌/核心设施/标志威胁）；同时上轮创建的 `07_道具图鉴.md` 可能存在与小说原文不完全对齐的条目。用户要求：**严格基于原小说**重写五大区+优化道具图鉴，精能大陆的 5 大分区框架仅作结构模板（不引入精能大陆专有名词与设定），道具图鉴应进一步对齐小说原文段落。

**用户反馈约束**：
- **严格按原小说来写**：所有内容必须可在 `帝王战队.txt` / `帝王战队番外.txt` / `小说指导.txt` 中找到对应段落
- **不引入精能大陆专有设定**：精能大陆的"8 堕落协会成员"（卜世/绝罗/泰特/雷克/灾朝/古洛/逐式/空碑）、"堕落教团"独立组织、"皇室/皇帝"等不得引入
- **精能大陆结构可借鉴**：5 大分区"风貌+核心设施+标志威胁"三段式作为模板
- **优化道具图鉴**：基于小说原文+模拟器条目，对 `07_道具图鉴.md` 进行二次优化

## What Changes
- **重写** `01_人类安全区.md` — 五大区（神耀/铁脊/凌海/黑渊/沐禾）严格基于小说
