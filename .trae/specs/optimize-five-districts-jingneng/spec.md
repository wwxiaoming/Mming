# 帝王战队 五大区 根据精能大陆 优化 Spec

## Why
当前 `/workspace/帝王战队资料/世界书条目v4/地区设定/01_人类安全区.md` 与 `02_魔神会控制区.md` 中的"五大区"仅是平铺的编号（一区/二区/三区/四区/五区），缺乏各自的风貌特色、核心设施、魔物威胁锚点。这种结构使 LLM 在游玩时无法生成差异化场景。`/workspace/精能大陆(小光).txt` 提供了"五大分区·特色与威胁锚点"成熟框架：凌海城/铁城/黑石城/沐禾城/星穹城，各有核心设施 + 标志威胁。需要将此差异化结构映射到帝王战队 五大区，并保留小说原文中实际提到的角色据点（一区皇后/暗狮、二区赫菲斯托斯/幽灵一号、三区 C 市/洛基+中心大剧院/文斌+奥西里斯等）。

## What Changes
- **重写** `01_人类安全区.md` — 为五大区分别赋予风貌特色、核心设施、标志威胁（参照精能大陆框架）

- **不修改** 任何 YAML 字段

### Requirement: 五大区各自风貌差异化
The new content SHALL 在 01_人类安全区.md 中明确呈现 5 大区各自独立的风貌特色、核心设施、魔物威胁锚点，参照精能大陆的"五大分区·特色与威胁锚点"结构。

**精能大陆映射参考**（仅作结构模板，小说角色据点优先）：
- **凌海城（东·沿海商业）** ↔ 帝王战队：**第三区 C 市**（联邦最大港口/贸易枢纽/精能研发高地，开放、多元、商业理性与奢侈消费并存）
- **铁城（北·重工业）** ↔ 帝王战队：**第二区**（重工业区，机械化束缚、效率至上的流水线）

- **沐禾城（南·农业森林）** ↔ 帝王战队：**第五区**（古老森林之地，生态扭曲、邪教仪式、自然共生型威胁）
- **星穹城（中央·行政）** ↔ 帝王战队：**第一区**（浮华天空之城/首都区，艺术化、高层阴谋、列阳世家所在）
