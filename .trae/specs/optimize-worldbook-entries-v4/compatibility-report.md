## 三、与角色卡的兼容性

### 3.1 帝王战队v1.58_优化版.json（272 keys）

- **V4 独立 keys 数**：413
- **重叠 keys 数**：80
- **重叠比例**：19%

**80 个重叠 key 列表**（节选 20）：
`世界观 / 乾坤 / 休 / 便器 / 光明之子 / 准星 / 凯风 / 分身 / 剑奴 / 千手魔罂粟 / 千瑞 / 反派 / 启润 / 嘉陵 / 圣灵 / 地下势力 / 堕落神装 / 大地之子 / 天火 / 天道之子 / ...`

**影响分析**：
- 重叠是**预期的**：v4 是帝王战队世界观的 Markdown 文档集，与 `帝王战队v1.58_优化版.json` 共享世界观、英雄、装置的核心 key
- 在酒馆中：v4 作为独立世界书 + 角色卡内嵌世界书同时激活时，**同一 key 会被两次触发**（来自两个不同来源）
- 这是**设计意图**：用户希望 v4 文档作为增强补充，而不是替代
- **建议**：在酒馆中只装载 v4 独立世界书，**不**装载 `帝王战队v1.58_优化版.json` 内的内嵌世界书，避免重复触发
