### 如何在 BRAIN 上构建 Alpha（合规方式）

BRAIN 平台提供两种构建 Alpha 的方式：

**1. Fast Expression（推荐入门）**

在平台上通过网页 UI 直接编写表达式，不需要任何本地环境：

1. 登录 BRAIN 平台
2. 点击 `Create Alpha` → 选择 `Fast Expression`
3. 选择数据集（Dataset）、时间范围、Universe 等参数
4. 编写 Alpha 表达式（例如：`rank(ts_delta(close, 5))`）
5. 点击 `Simulate` 进行回测
6. 查看回测结果，调整参数，反复迭代

**2. API 方式（进阶）**

BRAIN 平台提供 REST API，可以在本地用 Python 等语言编写脚本来提交和查询 Alpha：

- API 文档在平台 Developer 页面
- 需要申请 API Token
- 可以编写脚本批量测试不同参数组合
- **这是合规的自动化方式**，不同于 LQS 工具的作弊行为

## 三、Alpha 构建实操指导

### 3.1 注册流程

1. 打开 [worldquantbrain.com](https://worldquantbrain.com)
2. 点击 `Sign Up` 注册（或通过教程站的邀请链接注册，两者本质上都会绑定推荐关系）
3. 填写个人信息（姓名、邮箱、地区选择中国大陆）
4. 验证邮箱
5. **安装沉浸式翻译插件**（[immersivetranslate.net](https://www.immersivetranslate.net/)），BRAIN 平台全英文，浏览器自带翻译会导致页面报错

### 3.2 Fast Expression 语法入门

Alpha 表达式的基本结构：

```
算子(数据字段, 参数)
```

**常用算子示例：**

| 算子 | 含义 | 示例 |
|------|------|------|
| `rank(x)` | 横截面排名 | `rank(close)` — 按收盘价排名 |
| `ts_delta(x, d)` | d 天变化量 | `ts_delta(close, 5)` — 5 天涨跌幅 |
| `ts_mean(x, d)` | d 天均值 | `ts_mean(volume, 20)` — 20 日均量 |
| `ts_zscore(x, d)` | d 天标准化值 | `ts_zscore(returns, 10)` — 10 天 Z-score |
| `group_rank(x, group)` | 分组内排名 | `group_rank(pe_ratio, sector)` — 行业内市盈率排名 |

**构建步骤：**

1. 选择 Dataset（初学者推荐从 `analyst4` 或 `fundamental6` 开始）
2. 选择 Universe（TOP3000 适合入门）
3. 选择 Delay = 1（D1 模式, Sharpe 门槛 1.25）
4. 选择 Neutralization（初学推荐 MARKET 或 INDUSTRY）
5. 编写 Alpha 表达式
6. 点击 Simulate 回测

### 3.3 理解回测指标

提交 Alpha 需满足以下门槛（D1 模式）：

| 指标 | 门槛 | 含义 |
|------|------|------|
| **Sharpe** | > 1.25 | 衡量"每单位风险赚多少钱"。越高越好 |
| **Fitness** | > 1.0 | 综合适配度。受换手率、中性化等因素影响 |
| **Self-Correlation** | < 0.7 | 与你自己已提交 Alpha 的相似度。> 0.7 则不能提交。可以通过换数据集、换算子、换信号逻辑来降低 |
| **Turnover** | 1%-70% | 换手率。太高说明交易频繁（交易成本高），太低说明信号太弱 |

**颜色标记解读：**
- 🟢 绿色：通过自相关检查，可提交
- 🟡 黄色：相关性较高，需仔细检查指标
- 🔴 红色：未通过，直接放弃

### 3.4 每日提交流程与积分规则

**积分规则：**
- 每提交一个 Alpha 得 1,500-2,000 分（取决于质量等级）
- 日积分上限 2,000 分（所以每天最多提交 2 个即可）
- 结算时间：北京时间中午 12:00 结算提交数，14:00 结算积分
- 最快 5 天达到 10,000 分 → Gold 级别

**每日工作节奏：**
1. 上午：登录 BRAIN 平台，检查昨天的回测结果
2. 筛选绿色标记的 Alpha
3. 优先选 Sharpe 最高的 1-2 个提交
4. 写新的 Alpha 表达式、调整参数、重新回测
5. 下午 3 点后查看积分变化

**数据集+中性化推荐搭配（来自教程第 06 章，经社区验证）：**

| 阶段 | 数据集 | 中性化 | 预期产出 |
|------|--------|--------|----------|
| Day 1-2 | analyst4 | Subindustry | 1-2 个/天 |
| Day 3-4 | fundamental6 | INDUSTRY | 3-4 个/天 |
| Day 5-6 | fundamental2 | INDUSTRY / MARKET | 2-3 个/天 |

> 💡 如果连续两天没有产出，换数据集+中性化组合重新开始。
