# 第四章 免费领取云电脑

## 4.1 为什么需要云电脑
Alpha辅助工具需要24小时不间断运行，电脑不能休眠或关机。

## 4.2 推荐策略
阿里云3个月(第一选择) → 腾讯云1个月(第二选择) = 合计4个月免费
4个月足够成为Consultant

## 4.3 推荐配置
CPU 2核 / 内存 2GB / Windows Server 2022 Datacenter 中文版

## 4.4 连接方法
**方法一(推荐)**：Win+R → mstsc → 输入公网IP → 输入Administrator和密码
**方法二(手机)**：uu远程(uuyc.163.com)，云电脑和手机用同一账号登录
**方法三(紧急)**：阿里云控制台→轻量应用服务器→连接

## 4.5 环境配置
必须设置电源选项为"从不"休眠（控制面板→硬件和声音→电源选项）

## 4.6 安全建议
- 使用强密码
- 不要在服务器上浏览无关网页
- 定期更换远程桌面密码
- 远程桌面连接断开后程序继续运行，关闭窗口不等于关机
