# 云服务器配置

## 何时使用
当用户提到以下关键词时触发：云服务器、云电脑、阿里云、腾讯云、免费服务器、远程桌面、休眠、ECS、轻量应用服务器

## 核心指令
1. 阿里云申请流程：
   - 打开阿里云控制台 → 找到ECS（云服务器ECS）→ 免费试用入口
   - 选择配置：2核2GB、Windows Server 2022数据中心版
   - 完成实名认证后领取3个月免费试用
2. 腾讯云备选方案：
   - 前往腾讯云控制台 → 轻量应用服务器 → 免费试用
   - 配置：2核2GB、Windows Server 2022
   - 免费时长：1个月
3. 配置要求说明：最低2核2GB、推荐Windows Server 2022操作系统
4. 远程桌面连接方法：
   - 按Win+R → 输入mstsc → 输入服务器公网IP → 输入用户名Administrator和密码
   - 首次连接需接受证书提示
5. 手机操控方法：安装uu远程或其他远程桌面App，输入服务器IP和凭据
6. 关闭休眠设置（极其重要）：
   - 打开控制面板 → 电源选项 → 更改计划设置 → 将"使计算机进入睡眠状态"设为"从不"
   - 同时关闭显示器也建议设为"从不"
7. 切换服务器注意事项：工具配置需要更新新服务器IP、旧服务器到期前完成迁移
8. 安全建议：修改默认远程桌面端口、设置复杂密码、定期更新系统

## ⚠️ 警告
- 必须将休眠/睡眠设置为"从不"，否则工具无法持续运行
- 免费试用到期前如不续费，服务器数据将被清除，务必提前备份
- 不要把服务器密码设置为简单密码（如123456）
- 不要在服务器上访问不明网站或下载不明文件

## 💡 提示
- 阿里云ECS和腾讯云轻量应用服务器可以同时申请，叠加使用
- 推荐优先使用阿里云，免费期更长（3个月 vs 1个月）
- 如果免费试用名额用完，可以尝试换账号或等待下一批
- 服务器IP变更后记得更新工具的连接配置
