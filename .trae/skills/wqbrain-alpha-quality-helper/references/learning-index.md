# 学习资源索引

按场景分类的直达链接，配合 `wqbrain-alpha-quality-helper` 各子能力使用。

## 1. WQ 官方运算符字典

- 链接：https://platform.worldquantbrain.com/learn/operators
- 什么时候用：当 `expression-preflight` 提示你某个运算符用法有歧义、或你想换一个更稳的算子时，来这里查官方定义与示例。

## 2. WQ 官方数据集字典

- 链接：https://platform.worldquantbrain.com/learn/data-and-operators/datasets
- 什么时候用：当 `config-auditor` 提醒你某数据集在某区域缺数据、或换手 / Sharpe 异常时，来这里确认字段含义、覆盖区域和时间范围。

## 3. 老强教程主页

- 链接：https://alphadoc.biglongxia.com/
- 什么时候用：当你不确定"一阶段 / 二阶段 / 三阶段"分别该填什么参数，或想照搬老强的模板配置时，从这里找对应章节。

## 4. IMA 知识库（龙桑版）

- 链接：https://ima.qq.com/wiki/?shareId=f609041b0a72e8c20a8462b896618312c4625986e8d5c5ba1bfb00a43d239ae8
- 什么时候用：当 `result-diagnostician` 给出"为什么这个 alpha 在不同区域表现不一致"这类机理问题时，来这里找龙桑写的归因文章和复盘笔记。

## 5. 量化基础音频
