# WQ 平台提交硬性标准速查表

## 一、三硬性条件（提交门槛）

按论坛与官方共识，alpha 需同时满足以下三条才能成功提交并进入打分队列：

| 指标 | 阈值 | 说明 |
|---|---|---|
| **Sharpe（夏普比率）** | **> 1.25** | 风险调整后收益的硬性底线，低于此值直接被拒 |
| **Fitness（健康度）** | **> 1** | 综合考虑 Sharpe、收益与换手后的健康指标 |
| **Self Correlation（自相关性）** | **< 0.7** | 与平台已存在 alpha 的相关性，越低越能体现原创性 |

## 二、例外条款（高 Sharpe 豁免）

- 当新 alpha 的 **Sharpe 比历史相似 alpha 高出 10% 以上**（即 **Sharpe > 1.375**）时，可**忽略 Self Correlation 限制**。
- 适用场景：构造出明显强于历史均值的信号，但仍可能与若干旧 alpha 相似。
- 注意：Fitness 与 Sharpe 阈值仍然必须满足。

## 三、晋级路径（国内路径参考）

| 阶段 | 关键节点 | 说明 |
|---|---|---|
| 1. 起步 | **累计 10000 分 → 金牌（Gold）** | 通过持续提交达标 alpha 累积分数 |
| 2. 进阶 | **金牌 → 顾问邀请** | 平台顾问主动接触，受邀进入顾问群 |
| 3. 提速 | **累计 20+ 个合格 alpha** | 进入高产期，日产能显著提升 |
| 4. 常规 | **Regular：1-4 个 / 天** | 顾问权限下每日稳定产出 |
| 5. 顶级 | **满 100 个 Regular → 解锁 Super Alpha** | Super Alpha 权限：**每天 1 个**高阶提交额度 |

## 四、IQC 加分指标（提升入选概率）

| 指标 | 推荐阈值 | 作用 |
|---|---|---|
| **Sharpe** | **> 1.5** | 显著提高 IQC 入选率 |
| **Turnover** | **0.1 - 0.9**（约 10% - 90%） | 落入平台偏好的换手区间 |
| **Fitness** | **> 1** | 健康度达标 |
| **IC（信息系数）** | **> 0.02** | 与下一期收益的真实相关性 |

> 同时满足 IQC 指标的 alpha 有机会进入平台精选池，获得更高分值与曝光。

## 五、关键提醒（红线与注意事项）

- **严禁抄袭 alpha**：对其他用户 alpha 复制、轻微改字段、套壳提交均属违规，**永久封号**风险极高。
- **非原创检测**：平台对常见 alpha 模板和热门字段组合有查重机制，请务必在数据集、字段组合、信号逻辑上做出差异化。
- **每日分数上限 2000**：单日累计得分不超过 2000，超出部分不计入。
- **分数 T+1 日更新**：当日提交与得分情况在次日（北京时间上午左右）更新到账户。
- **中性化与分组**：多股票池、跨区域、跨行业的 alpha 在中性化与分组上需谨慎，避免因子拥挤。
- **回测区间**：建议覆盖至少一个完整牛熊周期，避免过拟合短期行情。

## 六、提交前自检清单

- [ ] Sharpe > 1.25（或 > 1.375 以触发例外条款）
- [ ] Fitness > 1
- [ ] Self Correlation < 0.7（或 Sharpe > 1.375 豁免）
- [ ] Turnover 落在合理区间（0.1 - 0.9 较优）
- [ ] Drawdown 未出现异常单股集中
- [ ] 字段、数据集、逻辑与历史 alpha 明显差异
- [ ] 未触及每日 2000 分上限
