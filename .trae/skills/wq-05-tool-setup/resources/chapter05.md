# 第五章 Alpha辅助工具的安装和使用

## 5.1 安装前确认
- WQ BRAIN和教程站账号已注册 ✅
- 云服务器已部署 ✅
- 记下了账号的邮箱和密码 ✅

## 5.2 安装工具
工具下载和上传详情见第4章。下载工具并上传到服务器。

## 5.3 工具使用要点
- 不需要自己下载Python，安装包已内置运行环境
- 首次运行时输入邮箱和密码登录，之后自动记住
- 工具需要连续运行，回测才会持续
- 安装后不需要配置任何额外东西

## 5.5 常见问题
- 工具一直转圈：网络问题，重新打开或检查连接
- 登录失败：检查密码是否区分大小写，先尝试登录官网确认账号密码正确
- 报错1001(请求太频繁)：IP被锁，需要导师后台解锁
- 防火墙拦截：关闭Windows防火墙和安全中心后重新下载
- WebView2安装失败：手动安装webview2环境
