# Alpha辅助工具安装使用

## 何时使用
当用户提到以下关键词时触发：工具、安装、辅助工具、提交Alpha、三线程、回测、工具卡住、工具报错

## 核心指令
1. 安装前确认：
   - 确认已经完成BRAIN平台注册
   - 确认已有云服务器（Windows Server 2022）
   - 确认服务器休眠已关闭
2. 安装流程：
   - 获取工具安装包（从社区或推荐人处获取）
   - 在云服务器上解压并运行安装程序
   - 按提示输入BRAIN平台账号信息
   - 完成初始配置（线程数、数据集选择等）
3. 如何用工具成功提交Alpha：
   - 在工具中选择数据集（Dataset）
   - 设置中性化（Neutralization）参数
   - 配置回测参数
   - 点击运行等待结果
   - 检查Alpha质量指标（Sharpe、Fitness、Self-Correlation、Turnover）
   - 符合标准的Alpha提交到平台
4. 官网生产和提交Alpha的方法：
   - 登录BRAIN平台 → Simulator/Alpha Editor
   - 手动编写或修改Alpha表达式
   - 运行回测（Simulation）
   - 检查指标通过后提交（Submit）
5. 三线程分配策略：
   - Day 1-2：单线程模式，使用analyst4 + Subindustry中性化组合
   - Day 3起：逐步增加到双线程，切换到其他数据集+中性化组合
   - 根据产出情况动态调整线程数和数据集组合
6. 常见问题处理：
   - 转圈卡住：检查网络连接，重启工具
   - 登录失败：确认账号密码正确，检查是否被封号
   - 并发已满：减少线程数，等待当前任务完成
   - 报错1001：检查工具版本是否过期
   - 回测无结果：更换数据集或中性化参数

## ⚠️ 警告
- 工具是非官方第三方工具，使用需自行承担风险
- 不要过度依赖工具，核心能力仍需自己掌握
- 工具配置错误可能导致Alpha全部无效
- 不要在官方社区或平台内讨论第三方工具
- 不要在社交媒体上分享工具界面截图

## 💡 提示
- 新手Day1-2先用单线程稳定产出，不要急于开多线程
- 工具运行期间不要操作服务器其他程序，避免资源竞争
- 每天检查工具的Alpha产出报告，及时调整策略
- 如有工具更新版本，及时升级但先备份配置
