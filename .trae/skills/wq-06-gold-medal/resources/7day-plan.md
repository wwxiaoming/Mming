# 7天第一阶段学习计划

## 第一天
1. 学习教程01-04，完成账户注册、工具安装，让工具跑起来
2. 加入WQ知识库，收藏小程序或网址
3. 在知识库提问：①什么是Alpha ②什么是量化 ③顾问薪酬结构 ④Genius Program等级与季度奖金
4. 继续把看不懂的名词提问给知识库

## 第二天
1. 中午12点前查看工具有无可提交Alpha，有则提交
2. 每天查看注册邮箱（官方邮件包括奖牌通知、课程预约、新人学习引导）
3. 如12点前提交了Alpha，下午3点后在官网查看积分
4. 听完播客《Alpha模型赚钱逻辑》（y.music.163.com/m/program?id=3719437212）
5. 在知识库提问：Long Count与Short Count的定义及用法

## 第三天
1. 中午12点前查看工具，提交Alpha
2. 下午3点后查看积分
3. 如有官方课程邮件，可预约听课（没时间也没事，有往期录播）
4. 听完播客《构建工业级量化Alpha》（y.music.163.com/m/program?id=3719480196）
5. 在知识库提问：什么是中性化（Neutralization）

## 第四天
1. 中午12点前查看工具，提交Alpha
2. 下午3点后查看积分
3. 重新学习教程06，向ima提问：什么是自相关性，如何降低
4. 如完成官方直播课，可尝试做课后作业（不做不影响成为顾问）

## 第五天
1. 中午12点前查看工具，提交Alpha
2. 下午3点后查看积分
3. 如果连续两天没有跑出可提交Alpha → 打开教程06，按官方推荐表更换数据集+中性化搭配重新跑

## 第六天（达成10000分后）
1. 先尽快提交满20个Alpha，之后每2天1个频率提交
2. 在ima帮助下学习教程07，在邮箱中找到基础测试问卷入口，完成测试——千万不要抄别人的答案！！
3. 在邮箱中找到Workday填写入口，按照教程08填写，务必仔细检查不要出错

## 第七天直到顾问审批通过
1. 先尽快提交满20个Alpha，之后每2天1个
2. 根据教程06官方推荐数据集+中性化和ima知识库，开始自我探索参数含义
3. 持续学习官方教程、论坛、Learn以及官方推荐公众号
4. ⚠️⚠️⚠️切记：不要在社群范围以外提到工具及教程站相关！！
5. 等待顾问审批流程通过

## ⚠️ 注意事项
- 达到10000分后官方一周内发邮件邀请成为顾问，继续多提交Alpha
- 一周后没收到邀请自行发邮件给support@worldquantbrain.com
- 达到10000分后极小概率触发账号冻结+面试(约3天准备时间)
