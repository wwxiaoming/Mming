# 第七章 填写基础测试问卷（Research Capability Test）

## 7.1 问卷概述
达到Gold等级后，WQ BRAIN平台会发送基础测试问卷链接。目的是验证你是否真正掌握了BRAIN平台的核心概念。

⚠️ 切勿直接复制他人答案！！平台使用机器审核系统，自动检测相似度，发现答案与他人高度雷同将直接被拒绝。
⚠️ 只有2次回答机会，第二次审核更严苛，务必一次通过！！

## 7.2 答题思路
1. 阅读参考资料尤其是平台Learn板块
2. 使用AI辅助理解核心概念
3. 用自己的话作答——每道题用自己的理解组织语言
4. 把答案与AI再次讨论，借助多个AI互相核查
5. 新建文档编辑答案——答题区显示框难用，建议新建文档答好再粘贴

⚠️ 填写到一半不会保存，不能暂停！

## 7.3 各题参考资料

### Q4: Matrix Data vs Vector Data
官方参考资料：platform.worldquantbrain.com/learn/documentation/understanding-data/data

### Q5: Long Count与Short Count
Long Count：策略建议做多（买入）的股票数量 = 模型预测价格上涨的标的数量
Short Count：策略建议做空（卖出）的股票数量 = 模型预测价格下跌的标的数量

用法：官方论坛参考support.worldquantbrain.com/hc/en-us/community/posts/11807866133911

### Q6: 中性化(Neutralization)
官方参考资料：platform.worldquantbrain.com/learn/documentation/create-alphas/neut-cons

### Q7: Alpha Example复现（开放题）
14个示例可选。答题模板：我复现了哪个Alpha → 它背后的市场逻辑 → 什么参数跑出什么结果 → 发现什么问题 → 为什么这样改 → 改完哪些指标变好哪些变差 → 最后理解

### Q8: 获取Alpha ID
步骤：打开Alpha页面 → 选择一个Alpha点开详情 → 点击"Open details in a new tab" → 浏览器地址栏最后一个标识符就是Alpha ID

### Q9: Genius计划级别与季度奖金
官方参考资料：support.worldquantbrain.com/hc/zh-cn/community/posts/27294185989527

## 7.4 注意事项
1. 用自己的语言作答——机器检测相似度
2. 理解而非记忆和生搬硬套
3. 善用Learn页面——官方文档最权威
4. 答案有条理——使用分点、分段
5. 适当使用英文术语
