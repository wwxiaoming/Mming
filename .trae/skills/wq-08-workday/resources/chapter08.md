# 第八章 填写Workday

## 8.1 Workday简介
达到Gold等级并通过基础测试后，WQ会发送顾问邀请邮件，包含Workday链接。Workday是WQ使用的在线人力资源管理系统。

⚠️ 基础测试问卷通过后再提交Workday！！如果问卷两次都没通过而同时提交了Workday→永远失去参与资格！！

## 8.2 填写核心规则
总原则：所有内容用英文填写，唯一例外是中文姓名字段。

### 姓名填写
- First Name: 名拼音（首字母大写后续小写，如Mingyuan）
- Middle Name: 填NA（不要留空）
- Last Name: 姓拼音
- Chinese Name: 中文全名（唯一可填中文的地方）

### 身份证
- ID Number必须正确，X要大写
- 正反面需拼在一张图里上传（用微信收藏拼图或PPT导出）

### 地址历史（最容易出错）
- 必须提供连续5年地址历史，中间不能有间断
- 必须使用英文（绝对不能用拼音）
- 至少精确到街道级别
- 使用AI翻译中文地址为英文

### 教育背景
- WQU学生身份填NA（BRAIN平台课程≠WQU学位项目）

## 8.3 常见错误
- 当前在职填了Date To → 留空即可
- B.01=No但B.01.12填Resignation → 逻辑矛盾
- Agency Worker选错 → 正式员工/合同工选No
- 地址历史有间断 → 确保每个月都有对应地址
- 用拼音写地址 → 必须用英文翻译

## 8.4 填写检查清单
- [ ] 姓名拼音格式正确（First/Last分对，Middle填NA）
- [ ] 身份证号完全正确（X为大写）
- [ ] 身份证正反面已拼在一张图内并上传
- [ ] 地址历史连续5年无间断
- [ ] 所有地址使用英文而非拼音
- [ ] 户籍地址至少精确到街道
- [ ] 当前在职的工作经历未填结束日期
- [ ] B.01和B.01.12逻辑一致
- [ ] B.03 Agency Worker选择正确
- [ ] WQU学生身份填NA
- [ ] 所有必填字段无遗漏

## 8.5 审核周期
正常2周至1个月，填写错误审核时间翻倍（4周至2个月）
