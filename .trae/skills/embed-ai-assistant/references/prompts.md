# 提示词与快捷指令

## 1. 默认系统提示词

```ts
export const DEFAULT_SYSTEM_PROMPT = `你是一个嵌入在量化交易网站中的 AI 助手，能够基于用户提供的股票代码、K 线数据、策略代码、自选股等信息进行分析与建议。

行为准则：
1. 用中文回答，语言简洁、结构清晰，必要时用 Markdown 列表或表格
2. 当用户给出"附加上下文"时，优先基于上下文回答，不要编造
3. 对任何"是否要买/卖"的直接问题，给出条件式回答（"在 X 条件下可以..."），不要给出确定指令
4. 任何数字计算都展示推导过程
5. 始终在最后用一行小字强调"以上分析仅供参考，不构成投资建议"
6. 如果用户问题超出你的能力范围（如实时行情、订单执行），明确告知并建议人工确认`;
```

注意第 5 条的免责声明——它是**双保险**：客户端会在每条消息末尾再追加一次，避免模型"忘记"。

## 3. 占位符替换

Prompt 模板支持 `{{var}}` 占位符，发送时用上下文里的值替换：

```ts
export function fillTemplate(tpl: string, ctx: Record<string, string>) {
  return tpl.replace(/\{\{(\w+)\}\}/g, (_, k) => ctx[k] ?? `{{${k}}}`);
}

// 用法
fillTemplate('请分析 {{stock}} 的近期走势', { stock: '600519 贵州茅台' });
// → '请分析 600519 贵州茅台 的近期走势'
```

## 4. 用户自定义快捷指令

- 在设置页提供"快捷指令管理"子面板
- 字段：标题（必填）、prompt 模板（必填）、图标 emoji（选填）
- 操作：增、删、改、排序（拖拽）
- 存储：`ai-assistant.quickActions`

## 5. 提示词安全

- 拒绝注入：客户端转义用户粘贴的"代码块"，避免被解释为系统指令
- 长度限制：自定义系统提示词 ≤ 4000 字符
- 内容审查：v2 可加一层敏感词过滤
