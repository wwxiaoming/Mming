# UI 设计参考

## 1. 抽屉（Drawer）

### 布局
- 宽度 360~420px，可拖拽改宽度（v2）
- 高度撑满视口，固定在右侧
- z-index 高于业务内容、低于全局 Modal
- 移动端：宽度 100%，更像底部 Sheet

### 动画
- 滑入：transform 200ms ease-out
- 关闭：transform 150ms ease-in
- 关闭时不卸载组件（保留消息），仅 `translateX(100%)`

### 交互
- Esc 关闭
- 点击抽屉外不关闭（避免误操作），仅右上角 × 关闭
- 抽屉状态写入 localStorage，下次进入页面保持

## 2. Header

```
┌─────────────────────────────────────────┐
│ [+] AI 量化助手       [DeepSeek ▾]  [+]│
│                            [新建] [⚙] [×]│
└─────────────────────────────────────────┘
```

- 标题左侧 + 号：打开/关闭抽屉
- 模型下拉：列出已配置的可用模型（也可"管理模型"）
- 新建：清空 messages，开始新会话
- ⚙：跳转到设置
- ×：关闭抽屉

## 3. MessageList

- **用户气泡**：右对齐，背景用主色调（蓝/紫），文字白色
- **助手气泡**：左对齐，背景深灰，文字浅色
- 头像：用户省略，助手用机器人 emoji 或图标
- **Markdown 渲染**：标题、列表、加粗、代码块、行内代码
- **代码块**：等宽字体 + 浅色背景 + 复制按钮
- **自动滚动**：新增消息时 `scrollTop = scrollHeight`；用户向上滚动查看历史时停止自动滚动（用 IntersectionObserver 判断"是否在底部"）
- **加载占位**：助手回复中显示三个跳动的小圆点
- **Disclaimer**：每条助手消息末尾追加 `⚠️ 以上分析仅供参考，不构成投资建议`

## 4. InputBar

```
┌─────────────────────────────────────────┐
│  [textarea                       ]      │
│                                         │
│                  120/500      [🚀 发送] │
└─────────────────────────────────────────┘
```

- 多行 textarea，min 1 行、max 6 行，超出后内部滚动
- Enter 发送、Shift+Enter 换行
- 实时显示 `当前字数 / 上限`
- 发送后按钮变 `[⏹ 停止]`，点击中断流
- 加载中 textarea 禁用
- 空消息禁止发送

- 用户进入对话后此区域隐藏

## 6. ContextPicker

点击"+"弹出 Popover：

```
┌─ 选择本轮附加上下文 ──────────┐
│ ☑ 当前股票：600519 贵州茅台  │
│ ☑ K 线摘要（最近 30 日）      │
│ ☐ 自选股列表                  │
│ ☐ 当前策略代码                │
│ ─────────────────────────── │
│              [取消] [插入]   │
└──────────────────────────────┘
```

- 选中的内容会序列化为文本，作为本轮 system 消息附在末尾
- 视觉上"本轮一次性"，不进历史

## 7. Settings（设置）

可以用整页路由也可以用 Modal，字段：

| 字段 | 类型 | 校验 |
|---|---|---|
| Base URL | input | 必填，URL 格式 |
| API Key | password (可显示) | 必填 |
| 模型名 | select / datalist | 必填 |
| Temperature | slider 0~2 | 默认 0.7 |
| 最大上下文轮数 N | number 1~50 | 默认 10 |
| 请求超时(秒) | number 5~300 | 默认 60 |
| 自定义系统提示词 | textarea | 选填，**保存前提示"覆盖默认免责"** |
| 请求模式 | radio: 直连 / 代理 | 默认直连 |

按钮：

- **测试连接**：调 `listModels()`，显示"成功，检测到 N 个模型"或具体错误
- **保存**：写 localStorage
- **恢复默认**：清空所有配置
