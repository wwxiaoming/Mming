# 第二章 FAQ — 常见问题汇总

## ⚠️ 踩坑速查表（11条）
1. 自行注册账号(⭐⭐⭐⭐) → 必须在助教导师引导下用邀请码注册
2. 抄袭问卷答案(⭐⭐⭐) → 用自己语言组织，理解后作答
3. 添加噪声降低Self-Correlation(⭐⭐⭐) → 更换数据集或改变信号逻辑
4. 地址用拼音填写(⭐⭐⭐⭐) → 用AI翻译成标准英文地址
5. 5年地址历史有空白(⭐⭐⭐⭐) → 确保时间线连续无间断
6. 在职工作填了截止日期(⭐⭐⭐⭐) → Date To留空表示当前在职
7. 离职原因与在职状态矛盾(⭐⭐⭐⭐) → B.01=No则B.01.12也填N/A
8. 服务器开代理/VPN(⭐⭐⭐) → 不需要科学上网，直连即可
9. 服务器进入休眠(⭐⭐⭐) → 电源选项设为"从不"休眠
10. 身份证银行卡信息不一致(⭐⭐⭐) → 信息必须一致
11. 忘记确认发票(⭐⭐⭐) → 账单日登录Workday手动确认

## (一)注册与资格
- 保险行业一般可加入，只要不是金融监管机构核心人员
- 无需金融/编程背景
- 不支持多账号，一人一号

## (二)分数与等级
- Challenge分用于达到Gold(10,000分)，IQC分是竞赛积分
- 无时间限制，每天最多2000分，最快5天
- VF(Value Factor)是平台综合评分，VF 0.95+收入可能是VF 0.5的十倍
- 等级升级：Gold(15+提交,3+Pyramid) → Expert(50+) → Master(200+) → Grandmaster(500+)

## (三)提交与优化
- D1提交标准：Sharpe>1.25, Fitness>1.0, Self-Correlation<0.7, Turnover 1%-70%
- D0(同日交易)：Sharpe需≥2.0
- 降低Self-Correlation：切换数据集、使用不同操作符、改变信号逻辑
- Performance Comparison：Change>0才提交，否则拖累整体组合

## (四)Workday与背调
- 地址必须用英文翻译，不能用拼音
- 身份证正反面需拼在一张图内上传
- 当前在职的Date To留空
- 无犯罪证明通常不需要

## (五)工具与服务器
- 工具需24小时连续运行，建议云服务器
- 不需要安装Python，安装包已内置运行环境
- Windows Server 2022推荐
- 免费4个月（阿里云3月+腾讯云1月）

## (六)收入与发薪
- 基础报酬每2月发一次(全年6次)，季度奖金每季度一次
- 每天只算前4个Alpha的Base Payment
- 已代扣代缴个税
