# 常见问题FAQ

## 何时使用
当用户提到以下关键词时触发：FAQ、常见问题、踩坑、报错、怎么办、为什么、问题、错误、解决

## 核心指令
1. 首先展示踩坑速查表（11条常见错误及正确做法和严重程度），包括但不限于：
   - 直接访问官网注册 → 必须通过推荐链接注册（严重）
   - Workday在问卷通过前提交 → 必须先通过问卷再提交Workday（致命）
   - 地址用拼音填写 → 必须用英文翻译（严重）
   - 金融行业选Yes → 必须选No（严重）
   - 身份证正反面分开上传 → 必须拼成一张图（中等）
   - 休眠设置不正确 → 必须设为"从不"（中等）
   - 提交Alpha后立即查看分数 → 需要等12点结算（轻微）
   - 使用不稳定的网络 → 使用稳定网络或云服务器（中等）
   - 翻译插件未安装 → 先安装沉浸式翻译插件（轻微）
   - 个人信息不真实 → 所有信息必须真实（致命）
   - 外传工具或链接 → 严格保密（严重）
2. 注册资格类问题（12问）：谁可以参加、地区限制、年龄要求、是否需要学历、是否需要金融背景、是否需要编程、是否需要工作经验、能否用手机、一个家庭多人注册、被拒后能否重新注册、注册费用、邀请码获取
3. 分数等级类问题（7问）：等级体系、积分计算、每日上限、结算时间、升级条件、降级规则、分数查看
4. 提交优化类问题（4问）：每天提交几个、提交时间选择、Alpha质量要求、无产出处理
5. Workday背调类问题（6问）：Workday填写时机、信息真实性、背调内容、背调时长、背调失败、银行卡要求
6. 工具服务器类问题（34问）：涵盖安装、配置、使用、故障排除
7. 收入发薪类问题（5问）：发薪周期、发薪方式、最低提现、纳税、收入查询

## ⚠️ 警告
- 踩坑表中的"致命"级别错误一旦发生可能导致永久失去资格
- FAQ答案基于当前政策，WorldQuant可能随时调整规则
- 涉及法律和税务问题建议自行咨询专业人士

## 💡 提示
- 优先查看踩坑速查表，大部分新手问题都能在此找到
- 遇到报错先搜索FAQ中是否有相关条目
- 如果FAQ未覆盖，建议在社区中提问但不要透露敏感信息
