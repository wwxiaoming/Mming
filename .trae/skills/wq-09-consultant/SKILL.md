# 成为顾问

## 何时使用
当用户提到以下关键词时触发：顾问、consultant、背调、背景调查、银行卡、收入开始、正式顾问、有条件顾问、面试

## 核心指令
1. 有条件顾问（Conditional Consultant）：
   - Workday提交后即可开始提交Alpha
   - 有条件顾问期间的Alpha计入Base Payment
   - 但如果背调不通过，期间收入作废
2. 背景调查（Background Check）：
   - 核实个人信息、学历、工作经历
   - 通常不需要无犯罪记录证明
   - 时长：几天到几周不等
   - 保持通讯畅通，及时响应背调机构的联系
3. 银行卡填写：
   - 在Workday中Bank Account Details模块
   - 银行名称填写英文（如ICBC、Bank of China）
   - 账号、Swift Code等信息准确填写
   - 银行卡必须为本人名下
4. 正式顾问（Full Consultant）：
   - 背景调查通过后自动成为正式顾问
   - Base Payment正常到账
   - 季度奖金按等级发放
5. 收入三类型：
   - Base Payment（基础支付）：按Alpha数量和质量计算的日常收入
   - Quarterly Bonus（季度奖金）：与等级挂钩
   - Referral Bonus（推荐奖金）：每成功推荐一人$100
6. 4大日收入影响因素（VF最重要）：
   - Value Factor（VF）：Alpha的价值因子，最关键指标
   - 提交数量：每天计前4个
   - 提交质量：Sharpe、Fitness等指标
   - 自我提升：持续优化策略
7. 准顾问提交策略：
   - 建议每2天提交1个Alpha保持账号活跃
   - 保留高质量Alpha等正式顾问后提交
   - 提交中间偏上质量的Alpha维持状态
8. 时间线总结：注册→学习→工具→产出→金牌→问卷→Workday→有条件顾问→背调→正式顾问→持续收入

## ⚠️ 警告
- 背调期间务必保持联系方式畅通
- Workday信息与背调结果不一致会导致背调失败
- 有条件顾问期间的收入取决于背调结果
- 银行卡必须是本人名下账户
- 不要在背调期间出现异常行为（如大量撤回Alpha）

## 💡 提示
- 有条件顾问期间不要过于激进地提交，保留实力
- 背调周期因人而异，耐心等待
- 银行卡信息提前向银行确认Swift Code
- 正式顾问后第一时间优化提交策略
