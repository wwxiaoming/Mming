# 股票分析技能整合文档

> 本文档整合 TRAE 中所有可用于股票分析、选股、调研、研报撰写的技能，构建从「数据获取 → 筛选 → 深度分析 → 报告输出」的端到端工作流。

---

## 一、技能全景图

| 类别 | 技能名称 | 核心定位 |
|------|----------|----------|
| 通用研报 | `consulting-analysis` | 咨询级研究报告生成（市场/品牌/行业/财务/竞品/投研尽调） |
| A股综合 | `a-share-analysis` | A股个股综合分析（基本面+技术面+政策+A股特性） |
| A股数据 | `a-stock-data` | A股全栈数据工具包（行情/研报/资金/新闻/公告/财务） |
| A股筛选 | `a-share-screener` | A股多策略筛选（价值/成长/动量/高分红） |
| A股价值 | `china-stock-analysis` | A股价值投资分析（akshare数据源，低频交易者友好） |
| 东财选股 | `eastmoney_select_stock` | 东方财富条件选股与板块成分股查询 |
| 价值投资 | `longbridge-value-investing` | 格雷厄姆/巴菲特价值投资（捡烟蒂+优质复利） |

| 供应链研究 | `serenity-skill` | 产业链卡点/瓶颈深度调研（投研尽调） |

## 三、核心工作流

### 工作流 A：单只个股深度调研

```
1. stock-watcher         → 加入观察池，确认跟踪名单
2. a-stock-data          → 拉行情/资金/公告/研报/财务全量数据
3. a-share-analysis      → 基本面+技术面+政策+A 股特性综合分析
4. longbridge-value-investing → Graham/Buffett 估值打分
5. investment-memo       → 输出投资备忘录（VC/PE/二级通用）
```

**适用场景**：你已经有目标股票，需要一份完整的尽调材料。

### 工作流 B：批量选股 + 多策略对比

```
1. a-share-screener      → 多策略批量筛选（价值/成长/动量/分红）
2. eastmoney_select_stock → 行业/板块成分股与指数筛选
3. china-stock-analysis  → 价值投资视角二次过滤
4. longbridge-value-investing → 捡烟蒂/优质复利双重打分
5. data-analysis         → 自定义财务数据深度分析
6. stock-watcher         → 入观察池持续跟踪
```

**适用场景**：从全市场挖掘符合特定策略的候选股票池。

### 工作流 C：产业链卡点调研（深度价值）

```
1. serenity-skill        → 产业链结构化梳理，寻找"断货即停产"环节
2. serenity-stock-choke  → 锁定卡脖子环节的小盘标的
3. a-stock-data          → 验证标的的研报/资金/筹码信号
4. last30days            → 确认海外/全球讨论热度与催化剂
5. consulting-analysis   → 产出产业链/竞品/投资尽调级报告
```

**适用场景**：基于"供应链瓶颈"逻辑挖掘未被充分定价的标的。

### 工作流 D：跟单研究（Serenity 风格）

```
1. serenity-radar        → 跟踪 Serenity 注意力轮动与新增标的
2. serenity-skill        → 对候选做产业链卡点压力测试
3. a-stock-data          → 数据交叉验证
4. investment-memo       → 形成自己的投资意见（不照搬）
```

**适用场景**：观察顶级独立研究者的思路，形成自己的投资框架。

### 工作流 E：行业/赛道研究 → 选股

```
1. consulting-analysis   → 行业研究框架 + 数据需求清单
2. last30days            → 海外/国内讨论热度与最新动态
3. eastmoney_select_stock → 锁定行业内上市公司
4. a-share-screener      → 行业内多维筛选
5. a-share-analysis      → 头部 + 中尾部公司逐一分析
6. investment-memo       → 形成赛道投资建议书
```

**适用场景**：自上而下，先看行业再选公司。

## 六、快速决策树

```
想做什么？
├─ 已有标的，要做尽调
│  └─ 工作流 A：单只深度调研
│
├─ 不知道买什么，要找标的
│  ├─ 有明确策略（价值/成长/动量/分红）
│  │  └─ 工作流 B：批量选股
│  ├─ 想找产业链机会
│  │  └─ 工作流 C：产业链卡点
│  ├─ 想跟单/对标
│  │  └─ 工作流 D：跟单研究
│  └─ 想看行业/赛道
│     └─ 工作流 E：行业研究
│
├─ 已有数据，要深度分析
│  └─ data-analysis (本地文件) + a-stock-data (补数据)
│
├─ 已有结论，要写报告
│  ├─ 投资备忘录 → investment-memo
│  └─ 咨询级研报 → consulting-analysis
│
└─ 跟踪持仓/自选
   └─ stock-watcher
```

## 八、使用建议

1. **从问题出发，不要从技能出发**：先用决策树定位你要解决的问题，再选工作流。
2. **数据交叉验证**：`a-stock-data` 拉数据后，可用 `eastmoney_select_stock` 或 `data-analysis` 二次核对。
3. **方法论先于结论**：做选股前先用 `longbridge-value-investing` 或 `china-stock-analysis` 确定框架，避免被数据噪音带偏。
4. **报告分层输出**：
   - 内部速记 → `stock-watcher` 摘要
   - 投资建议 → `investment-memo`
   - 行业/竞品研究 → `consulting-analysis`
5. **风险提示**：所有技能仅提供研究与决策支持，不构成交易建议；A股 T+1、涨跌幅限制、北向资金等特性已在 `a-share-analysis` 中内置考量。
