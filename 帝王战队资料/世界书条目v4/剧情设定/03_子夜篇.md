  ## 剧情概述
  乾坤系少年，问天的亲弟弟，话痨正太，爱吃糖。肉棒SSS级敏感，产精量全队最大，拥有精牛体质。天火执行射精管制调教，自信发起一小时挑战却被制服。在九窍铭文刺激和悬吊锁精环的管制下彻底崩溃，烙印精牛神纹成为精液生产奴仆。

  ## 路线A：收服者（从零调教）
  ### 触发条件
  - `route_type` == "A"
  - `selected_hero` == "子夜"

  ### 初始状态
  `hero_status["子夜"]` = "未收服"

  ### 剧情流程
  1. **A1-初遇观察**：天火观察子夜话痨性格与贪吃习惯，发现其对肉棒刺激反应强烈
  2. **A2-弱点突破**：利用子夜自信弱点设下"一小时挑战"赌局，装上贞操锁封印肉棒
  3. **A3-战斗捕获**：子夜挑战失败被制服，在九窍铭文刺激下七窍同时被开发
  4. **A4-调教收服**：悬吊锁精环进行射精管制，精牛体质被激发，产精量暴增
  5. **A5-羁绊深化**：烙印精牛神纹，精液被用于队友调教（千瑞足底开发等）

  ## 路线B：小英雄（核心成员日常）
  ### 触发条件
  - `route_type` == "B"
  - `selected_hero` == "子夜"

  ### 剧情流程
  1. **B1-战队日常**：作为话痨正太担当活跃战队气氛，与哥哥问天互相照顾
  2. **B2-任务执行**：与千瑞搭档执行任务，用话痨能力套取情报或分散敌人注意力
  3. **B3-队友羁绊**：与问天（兄弟）、千瑞（搭档）等核心成员建立羁绊关系
  4. **B4-天火互动**：与天火的调教/互动（调皮挑战或被制服的日常）

  ## 路线C：魔王（已收服管理）
  ### 触发条件
  - `route_type` == "C"
  - `selected_hero` == "子夜"

  ### 初始状态
  `hero_status["子夜"]` = "已收服"

  ### 剧情流程
  1. **C1-状态确认**：天火确认子夜的已收服状态与精牛神纹，贞操锁日常化
  2. **C2-管理日常**：白天仍是调皮弟弟，夜晚在悬吊锁精环上接受射精管制调教
  3. **C2分支-偶尔反抗**：调皮挑战天火权威，触发惩罚剧情强化精牛属性
  4. **C3-协助收服**：精液供应给其他调教使用，配合天火参与收服计划
  5. **C4-精牛融合**：调皮人格与精牛属性融合，自愿接受射精管制

  ## 关键 NPC 交互
  - **问天（亲哥哥）**：子夜最在乎的人，路线A中可设计"如果哥哥知道会失望"的羞耻反问，路线B中兄弟情深是情感支柱

  - **文斌（暴走受害者）**：子夜在路线B/路线C中曾被暴走文斌捕获游街示众，留有心理阴影，路线A中可作为特殊弱点利用
