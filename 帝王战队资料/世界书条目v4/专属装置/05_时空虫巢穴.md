  ## 装置概述

  时空虫巢穴是专为凯风设计的专属榨精装置，针对其时空腺体SSS级敏感的独特体质。装置外形如巨大的虫巢，内部充满特化时空虫群。少年四肢被时空触须拉开悬吊于虫池中央，通过环境压制制造腺体饥渴，配合龟头真空啜吸虫、茎身蠕动榨精套、腺体协同刺激三大核心系统，实现时空腺体的全面榨取。

  - **类型**：时空虫群 + 腺体饥渴榨精装置
  - **改造原型**：虫巢生态舱
  - **适用英雄**：凯风（时空腺体 SSS、时之卵、空之球）
  - **存放位置**：神魔号独立舱室 07
  - **设计目标**：以"环境压制制造饥渴"为特色，瓦解凯风的高傲姿态

  ## 固定方式

  ### 四肢悬吊
  - 四肢被时空触须拉开悬吊于虫池中央
  - 双腿大张呈倒 V 形，阴茎勃起朝上完全暴露
  - 双手被拉开固定于两侧，无法遮挡或保护

  ### 触须束缚
  - 时空触须由时空魔力凝结而成
  - 触感柔软但束缚力极强
  - 可根据挣扎程度自动调整束缚力度

  ### 姿态特点
  - 悬吊姿态让少年完全无助
  - 全身敏感带暴露于虫群视野
  - 无法躲避任何形式的刺激
  - **联动效果**：悬吊 + 虫群包围 + 饥渴制造 = 高傲被完全剥夺

  ## 核心组件

  本装置包含 5 个核心模块，每模块含**结构 + 触发逻辑 + 与敏感度的联动**：

  ### 组件 1：环境压制系统（饥渴制造模块）

  - **结构**：
    - 池中时空虫群大幅降低时空元素浓度
    - 制造时空腺体的饥渴状态
    - 长期未刺激的空虚感被强制激活
  - **触发逻辑**：
    - 启动后持续抽离时空元素
    - 检测到腺体空虚度后调整浓度
  - **与敏感度的联动**：
    - 睾丸（时之卵）与前列腺（空之球）产生强烈渴求
    - 被触碰时渴求如决堤般爆发
    - 高傲的凯风最羞于承认的弱点被精准打击

  ### 组件 2：龟头真空啜吸虫（龟头特化模块）

  - **结构**：
    - 数只特化时空虫完整包裹龟头
    - 虫体腔内柔软吸盘与细密绒毛
    - 专门针对龟头敏感带设计
  - **触发逻辑**：
    - 持续啜吸：稳定负压吸附
    - 旋转刮磨：虫体旋转刺激龟头表面
    - 脉冲啜吸：快速吸放交替
  - **与敏感度的联动**：
    - 真空啜吸与旋转刮磨同步进行
    - 重点责磨冠沟与系带
    - 模拟多个口腔同时吮吸龟头

  ### 组件 3：茎身蠕动榨精套（茎身活体模组）

  - **结构**：
    - 时空虫分泌活体套膜包裹阴茎茎身
    - 套膜紧密贴合阴茎表面
    - 可随阴茎勃起程度自动调整
  - **触发逻辑**：
    - 模拟喉腔节律性蠕动
    - 从根部至龟头交替收缩榨取
    - 形成波浪式挤压效果
  - **与敏感度的联动**：
    - 套膜分泌润滑粘液
    - 同时含有轻微致敏成分提升敏感度
    - 茎身被蠕动套榨取形成立体包裹

  ### 组件 4：腺体协同刺激系统（核心榨取模块）

  - **结构**：
    - 虫腹足包裹睾丸（时之卵）
    - 触手状口器深入后庭（空之球）
    - 其余虫群覆盖全身舔舐
  - **触发逻辑**：
    - 变频振动按摩时之卵
    - 注入催情粘液刺激空间魔力
    - 舔舐足心、乳头等敏感带
  - **与敏感度的联动**：
    - **时之卵刺激**：刺激时间魔力的储存与转化
    - **空之球刺激**：缠绕挤压前列腺，注入催情粘液
    - **全身协同**：分泌神经致敏粘液提升全身敏感度

  ### 组件 5：高傲崩坏系统（心理打击模块）

  - **结构**：
    - 装置内置语音播报系统
    - 可调用凯风"嘴硬"言论的录音
    - 镜子显示凯风被榨精的实时画面
  - **触发逻辑**：
    - 在凯风高潮时自动播报其高傲言论
    - 与当前被榨精的狼狈状态形成对比
  - **与敏感度的联动**：
    - 越是平时嘴硬说风凉话
    - 崩溃时反差越剧烈
    - 被触手调教的记忆被反复唤醒

  ## 核心机制

  本装置设 4 种状态模式，每模式含**触发条件 + 执行流程 + 终止条件**：

  ### 状态 1：压制模式（饥渴制造）

  - **触发条件**：装置启动后默认进入
  - **执行流程**：
    - 时空元素浓度降低
    - 腺体开始产生饥渴感
    - 少年在空虚中挣扎
  - **终止条件**：腺体饥渴度达阈值（→ 激活模式）

  ### 状态 2：激活模式（饥渴爆发）

  - **触发条件**：腺体饥渴度达标
  - **执行流程**：
    - 虫群开始接触敏感带
    - 饥渴感转化为强烈渴求
    - 被触碰时快感爆发
    - 高傲崩坏系统开始对比播报
  - **终止条件**：检测到接近榨取阈值（→ 榨取模式）

  ### 状态 3：榨取模式（全面榨精）

  - **触发条件**：榨取阈值突破
  - **执行流程**：
    - 腺体被全面刺激
    - 时之卵与空之球协同运作
    - 精液大量产出
    - 全身协同舔舐
  - **终止条件**：射精完成（→ 恢复模式）

  ### 状态 4：恢复模式（循环准备）

  - **触发条件**：射精完成
  - **执行流程**：
    - 短暂休息后重新开始压制
    - 腺体重新进入饥渴状态
    - 恢复体力
  - **终止条件**：饥渴度回升（→ 循环至激活模式）

  ### 联动机制

  - **饥渴-榨取闭环**：压制→饥渴→激活→榨取→恢复→压制
  - **腺体协同**：时之卵 + 空之球双腺体同步运作
  - **高傲崩坏反馈**：语音播报 + 实时画面 + 记忆唤醒
  - **多虫协同**：龟头虫 + 茎身套 + 腺体虫 + 全身虫群
