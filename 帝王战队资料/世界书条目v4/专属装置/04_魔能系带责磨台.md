  ## 装置概述

  魔能系带责磨台是专为阚泽设计的专属榨精装置，针对其魔能系带SSS级敏感的独特体质。装置采用跪缚姿态固定，使阴茎腹侧朝上，魔能系带完全暴露。通过系带滚轮刮擦器、系带真空吸盘组、龟头冠状沟夹揉环、阴茎茎身搓揉套四大核心组件的协同运作，实现针对魔能系带的精准高强度刺激与魔力糖液回收。

  - **类型**：跪缚责磨 + 魔能糖液回收装置
  - **改造原型**：刑讯台/榨糖机
  - **适用英雄**：阚泽（魔能系带 SSS）
  - **存放位置**：神魔号独立舱室 06
  - **设计目标**：以"糖液回收"为特色，榨取阚泽的魔能系带产物

  ## 固定方式

  ### 跪缚姿态
  - 少年跪于责磨台中央的软质跪垫上
  - 双腿被分开固定，大腿与小腿呈 90 度弯曲
  - 腰部被束带固定，上身微向前倾

  ### 阴茎暴露
  - 阴茎腹侧朝上，魔能系带完全暴露
  - 阴茎被支架托起固定，无法躲避刺激
  - 龟头与冠状沟同样完全暴露

  ### 手部束缚
  - 双手被反绑于背后
  - 无法遮挡或保护敏感区域
  - 被迫接受持续的系带刺激

  ### 姿态特点
  - 跪缚姿态既羞耻又无助
  - 魔能系带暴露于视野中央
  - 满足"被强制命令"的服从快感
  - **联动效果**：跪姿+暴露系带+命令系统 = 阚泽的"被制服"XP 全面激活

  ## 核心组件

  本装置包含 5 个核心模块，每模块含**结构 + 触发逻辑 + 与敏感度的联动**：

  ### 组件 1：系带滚轮刮擦器（系带责磨模块）

  - **结构**：
    - 微型软质滚轮贴合系带表面
    - 表面仿舌苔颗粒，模拟猫舌舔舐的粗糙刮蹭感
    - 可沿系带全长往复滚动
  - **触发逻辑**：
    - 慢速模式：温柔刮擦，延长快感
    - 快速模式：高频刮蹭，快速逼近高潮
    - 脉冲模式：间歇性加强刺激
  - **与敏感度的联动**：
    - 滚轮从尿道口下方至冠状沟整条敏感带
    - 覆盖系带 SSS 神经密集区域
    - 每一寸系带皮褶都被细致照顾

  ### 组件 2：系带真空吸盘组（糖液回收模块）

  - **结构**：
    - 沿系带排列三个微型吸盘（上/中/下段）
    - 吸盘连接回收管道
    - 可独立或协同运作
  - **触发逻辑**：
    - 交替负压吸附与释放
    - 拉拽系带皮褶同时刺激皮下密集神经末梢
  - **与敏感度的联动**：
    - 吸附拉拽系带皮褶形成反复吮吸的快感
    - **核心功能**：同步回收渗出的魔力糖液
    - 糖液可用于能量补充或进一步调教

  ### 组件 3：龟头冠状沟夹揉环（龟头联动模块）

  - **结构**：
    - C 型软夹自龟头背侧卡入冠状沟
    - 内置偏心振动球
    - 紧贴系带与冠沟交界处
  - **触发逻辑**：
    - 三种振动模式：持续 / 脉冲 / 渐变
    - C 型夹可调节夹持力度
  - **与敏感度的联动**：
    - 偏心振动球持续揉压冠沟 SS 敏感点
    - 与系带刺激形成协同效应
    - "上下夹击"形成系带-冠沟联动

  ### 组件 4：阴茎茎身搓揉套（茎身协奏模块）

  - **结构**：
    - 硅胶套包裹阴茎中后段
    - 内壁配备可旋转搓揉结构
    - 模拟手掌交错扭搓
  - **触发逻辑**：
    - 双向旋转搓揉阴茎茎身
    - 强度可与系带刺激同步
  - **与敏感度的联动**：
    - 与系带刺激形成上下双重快感叠加
    - 茎身与系带同时被刺激
    - 快感从多方向同时涌入

  ### 组件 5：服从命令系统（XP 触发模组）

  - **结构**：
    - 装置内置语音播报系统
    - 可接入外部命令音频
    - 支持编号管理与状态报告
  - **触发逻辑**：
    - 装置可播报"9527 号罪犯向长官报道"
    - 语音命令控制装置运作
  - **与敏感度的联动**：
    - 强化被强制命令的服从快感
    - 满足阚泽的"热血被制服"XP
    - 语音播报与身体刺激形成"听触"协同

  ## 核心机制

  本装置设 4 种状态模式，每模式含**触发条件 + 执行流程 + 终止条件**：

  ### 状态 1：预热模式（低速唤醒）

  - **触发条件**：装置启动后默认进入
  - **执行流程**：
    - 滚轮刮擦器低速运转
    - 系带表面被温柔刮蹭
    - 魔力糖液开始渗出
  - **终止条件**：糖液渗出率达阈值（→ 加强模式）

  ### 状态 2：加强模式（糖液加速）

  - **触发条件**：糖液渗出率达标
  - **执行流程**：
    - 真空吸盘组开始运作
    - 吸附拉拽系带皮褶
    - 糖液渗出量增加
    - 茎身搓揉套开始辅助
  - **终止条件**：糖液渗出量持续上升（→ 协同模式）

  ### 状态 3：协同模式（全组件激活）

  - **触发条件**：糖液量已显著
  - **执行流程**：
    - 冠状沟夹揉环启动振动
    - 茎身搓揉套开始搓揉
    - 所有组件协同刺激
    - 命令系统播报"9527 号"
  - **终止条件**：检测到接近射精（→ 高潮模式）

  ### 状态 4：高潮模式（榨糖液）

  - **触发条件**：射精阈值突破
  - **执行流程**：
    - 所有组件同步加强
    - 系带被极限刺激
    - 糖液大量渗出伴随射精
    - 真空吸盘高速回收糖液
  - **终止条件**：糖液回收完毕（→ 循环至预热模式）

  - **糖液-射精同步**：糖液渗出量与射精阈值联动
