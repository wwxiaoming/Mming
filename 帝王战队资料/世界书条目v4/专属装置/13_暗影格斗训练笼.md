  ## 装置概述

  暗影格斗训练笼是专为文斌设计的专属榨精装置（亦为战斗训练装置），针对其"与目标同归于尽"的战斗风格与"嫉妒兄弟"的深层心理。装置由八角笼格斗场改造而来，外壁由暗影触须构成的触手群构成。装置模拟一个"会主动攻击的真人"，在文斌将其击败时，装置会通过触手进行"色情式"奖励。文斌的誓约能力、弱点洞悉、暴走状态均可在此装置中模拟训练。

  - **类型**：战斗+调教联用装置
  - **改造原型**：八角笼格斗场
  - **适用英雄**：文斌
  - **存放位置**：神魔号独立舱室 15
  - **设计目标**：以"战斗胜利→触手奖励"为闭环，将文斌的战斗力转化为榨精动力

  ## 固定方式

  ### 站立格斗
  - 少年站立于八角笼中央
  - 双手不固定（可自由战斗）
  - 双脚不固定（可自由移动）

  ### 触手包围
  - 笼壁由暗影触须构成的触手群构成
  - 触手可主动攻击
  - 模拟"会主动攻击的真人"

  ### 战斗模拟
  - 装置可模拟各种敌人
  - 文斌的"誓约能力"可在此训练
  - 文斌的"弱点洞悉"可在此激活

  ### 姿态特点
  - 站立格斗姿态让文斌保持战斗状态
  - 触手包围强化"被多个敌人包围"的紧张感
  - 战斗胜利触发触手奖励
  - **联动效果**：战斗姿态 + 触手包围 + 战斗胜利奖励 = 战斗欲望与榨精欲望深度绑定

  ## 核心组件

  本装置包含 5 个核心模块，每模块含**结构 + 触发逻辑 + 与敏感度的联动**：

  ### 组件 1：暗影触须触手群（触手墙模块）

  - **结构**：
    - 笼壁由数千条暗影触须构成
    - 触手可模拟多种攻击方式
    - 可切换为仿真人形（模拟真人战斗）
  - **触发逻辑**：
    - 启动后触手群开始攻击
    - 文斌击败目标时触手变形
  - **与敏感度的联动**：
    - 触手在攻击中接触文斌皮肤
    - 战斗胜利时触手切换为色情模式
    - 形成"战斗→奖励"闭环

  ### 组件 2：仿真人形模拟系统（真人模拟模块）

  - **结构**：
    - 触手群可重构为仿真人形
    - 模拟各种敌人或天火
    - 可定制人形的能力与外观
  - **触发逻辑**：
    - 启动后触手群变形
    - 形成仿真人形
    - 文斌与之战斗
  - **与敏感度的联动**：
    - 击败仿真人形后触手群切换为色情奖励
    - 仿真人形可以是天火或兄弟
    - 形成"击败目标=获得奖励"的心理绑定

  ### 组件 3：誓约能力训练系统（誓约训练模块）

  - **结构**：
    - 模拟目标显示文斌需凝视的"眼睛"
    - 装置播报誓约词
    - 模拟目标释放能力，文斌需誓约
  - **触发逻辑**：
    - 启动后模拟目标出现
    - 文斌使用誓约能力
    - 装置反馈誓约效果
  - **与敏感度的联动**：
    - 誓约时文斌消耗大量魔力
    - 誓约后装置自动进入色情奖励
    - 形成"誓约=榨精准备"的心理暗示

  ### 组件 4：弱点洞悉激活系统（弱点分析模块）

  - **结构**：
    - 模拟目标具有"隐藏弱点"
    - 文斌需通过弱点洞悉发现
    - 弱点分析结果以直觉形式呈现
  - **触发逻辑**：
    - 启动后模拟目标暴露部分弱点
    - 文斌深度分析
    - 装置反馈完整弱点
  - **与敏感度的联动**：
    - 弱点洞悉激活时文斌进入暴走
    - 暴走状态易激发情欲驱动
    - 形成"分析弱点=情欲驱动"的心理绑定

  ### 组件 5：嫉妒心理触发系统（XP 强化模块）

  - **结构**：
    - 装置模拟天火与"其他人"亲密场景
    - 播放"文斌是失败者"等刺激性语音
    - 模拟文斌"输给了兄弟"的情境
  - **触发逻辑**：
    - 启动后嫉妒触发场景显示
    - 文斌战斗欲望被强化
    - 战斗胜利后的奖励更激烈
  - **与敏感度的联动**：
    - 嫉妒心理强化战斗动力
    - 战斗胜利后触手奖励更激烈
    - 形成"嫉妒→战斗→触手奖励"循环

  ## 核心机制

  本装置设 4 种状态模式，每模式含**触发条件 + 执行流程 + 终止条件**：

  ### 状态 1：战斗训练模式（基础训练）

  - **触发条件**：装置启动后默认进入
  - **执行流程**：
    - 仿真人形出现
    - 文斌与之战斗
    - 使用誓约能力
    - 弱点洞悉激活
  - **终止条件**：文斌击败仿真人形（→ 奖励模式）

  ### 状态 2：触手奖励模式（色情反馈）

  - **触发条件**：文斌击败仿真人形
  - **执行流程**：
    - 仿真人形变形为触手
    - 触手群开始色情式接触
    - 文斌的阴茎、睾丸、肛门被刺激
  - **终止条件**：天火指令（→ 继续战斗或结束）

  ### 状态 3：暴走模式（弱点洞悉激活）

  - **触发条件**：文斌深度分析弱点
  - **执行流程**：
    - 文斌进入暴走状态
    - 情欲驱动被激发
    - 弱点洞悉更精准
    - 战斗更激烈
  - **终止条件**：暴走状态结束（→ 战斗结束模式）

  ### 状态 4：战斗结束模式（榨精释放）

  - **触发条件**：暴走状态结束或战斗完成
  - **执行流程**：
    - 触手群集中榨取
    - 文斌被榨精
    - 精液被收集
    - 战炎、暮黎被榨精画面同步显示
  - **终止条件**：天火指令（→ 恢复）

  ### 联动机制

  - **战斗-奖励闭环**：战斗 → 胜利 → 触手奖励
  - **誓约-榨精闭环**：誓约消耗魔力 → 色情奖励 → 补魔
  - **弱点-情欲闭环**：弱点洞悉 → 暴走 → 情欲驱动 → 榨精
  - **嫉妒-战斗闭环**：嫉妒触发 → 战斗动力 → 战斗激烈 → 奖励更激烈

  - 弱点洞悉激活时情欲驱动涌现
  - 嫉妒心理强化战斗动力

  ### 长期效果
  - 战斗欲望与榨精欲望深度绑定
  - 对击败仿真人形产生条件反射
  - 弱点洞悉时易进入情欲驱动
  - 嫉妒心理成为榨精的隐性动机

  ### 心理效果
  - "战斗胜利=获得奖励"的心理印记
  - "嫉妒兄弟"成为战斗动力
  - 在战斗+榨精中找到扭曲的双重满足

  - **路线 A 收服者**：文斌初榨时强抵抗"战斗与榨精绑定"；后期主动要求启动装置

  - **路线 C 已收服**：文斌主动要求与仿真人形战斗+榨精
