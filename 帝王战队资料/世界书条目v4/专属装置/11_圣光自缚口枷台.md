  ## 装置概述

  圣光自缚口枷台是专为战炎设计的专属榨精装置，针对其"主动献祭以维系光明"的深层心理。装置将战炎固定于高台之上，以其自身圣光能量为燃料，通过自缚机制将圣光反噬。当战炎试图挣脱时，圣光会反噬自身体内形成极致快感，使其在"主动献祭"与"被动快感"之间形成矛盾循环，配合暮黎的兄弟羁绊作为催化剂，达成"光明被玷污"的核心心理打击。

  - **类型**：圣光自缚 + 献祭榨精装置
  - **改造原型**：圣光祭坛
  - **适用英雄**：战炎（圣光系 S+）
  - **存放位置**：神魔号独立舱室 13
  - **设计目标**：以"自缚献祭"为特色，将战炎的"主动献祭"心理转化为榨精动力

  ## 固定方式

  ### 高台站立
  - 少年被固定于高台之上
  - 双腿被分开站立于高台两侧
  - 双手被固定于胸前（祈祷姿态）
  - 上身被固定带锁定

  ### 圣光束缚
  - 战炎自身的圣光能量被束缚系统抽离
  - 圣光形成锁链缠绕全身
  - 锁链可由战炎自身控制（自缚特性）

  ### 暮黎羁绊
  - 暮黎（弟弟）的虚弱状态实时显示
  - 战炎通过圣光链接感知暮黎
  - 暮黎的求助成为心理压力源

  ### 姿态特点
  - 站立祈祷姿态增加"献祭"的仪式感
  - 自缚机制让战炎无法将责任推给他人
  - 暮黎的求助增加心理负担
  - **联动效果**：高台姿态 + 自缚 + 暮黎 = "为救弟弟而献祭自身"的心理闭环

  ## 核心组件

  本装置包含 5 个核心模块，每模块含**结构 + 触发逻辑 + 与敏感度的联动**：

  ### 组件 1：自缚圣光锁链系统（核心束缚模块）

  - **结构**：
    - 圣光形成的锁链可由战炎自身控制
    - 锁链强度与战炎圣光储量正相关
    - 战炎主动收紧=献祭程度加深
  - **触发逻辑**：
    - 启动后战炎可主动操控锁链
    - 每次主动收紧触发圣光反噬
  - **与敏感度的联动**：
    - 圣光反噬形成极致快感
    - 战炎无法将责任推给他人
    - 形成"主动献祭=获得快感"的矛盾循环

  ### 组件 2：口枷强制口交系统（榨精模块）

  - **结构**：
    - 大型口枷将口腔完全撑开
    - 装置自带震动舌头与吮吸球
    - 模拟强制口交的场景
  - **触发逻辑**：
    - 启动后口枷撑开
    - 舌头与吮吸球开始运作
    - 模拟天火肉棒插入的体验
  - **与敏感度的联动**：
    - 口腔与喉部敏感被持续刺激
    - 模拟口交场景强化"被主人榨取"心理
    - 与自缚系统形成"主动献祭+被动榨取"闭环

  ### 组件 3：圣光反噬榨精系统（核心榨取模块）

  - **结构**：
    - 装置抽离战炎圣光能量
    - 反噬时圣光转化为榨精能量
    - 战炎自身的光明力量成为榨精工具
  - **触发逻辑**：
    - 启动后缓慢抽离圣光
    - 战炎挣扎时加速反噬
  - **与敏感度的联动**：
    - 圣光反噬形成极致快感
    - 越挣扎越被榨
    - 形成"光明被玷污"的心理打击

  ### 组件 4：暮黎羁绊催化系统（心理放大模块）

  - **结构**：
    - 暮黎（弟弟）状态实时显示
    - 暮黎的求助声可定时播放
    - 暮黎被榨精的画面可投影
  - **触发逻辑**：
    - 启动后暮黎状态显示
    - 战炎自缚时暮黎被榨精画面切换
  - **与敏感度的联动**：
    - 暮黎的求助增加心理压力
    - 兄弟羁绊的"为救弟弟"成为献祭动机
    - 形成"为弟弟献祭=榨取自身"的矛盾逻辑

  ### 组件 5：榨精采集与展示系统（数据反馈模块）

  - **结构**：
    - 口枷连接的精液收集器
    - 数据实时显示
    - 历史成绩记录
  - **触发逻辑**：
    - 启动后收集精液
    - 实时显示榨精量
  - **与敏感度的联动**：
    - 数据反馈让战炎持续榨取
    - "为救弟弟"的使命感被数据化
    - 形成"榨精即救人"的心理扭曲

  ## 核心机制

  本装置设 4 种状态模式，每模式含**触发条件 + 执行流程 + 终止条件**：

  ### 状态 1：预缚模式（圣光抽离）

  - **触发条件**：装置启动后默认进入
  - **执行流程**：
    - 圣光锁链缠绕战炎
    - 缓慢抽离圣光能量
    - 战炎感到自身光明减弱
  - **终止条件**：圣光被抽离至反噬阈值（→ 自缚献祭模式）

  ### 状态 2：自缚献祭模式（核心榨取）

  - **触发条件**：圣光被抽离至反噬阈值
  - **执行流程**：
    - 战炎可主动操控锁链收紧
    - 圣光反噬形成极致快感
    - 口枷模拟强制口交
    - 暮黎求助声定时播放
  - **终止条件**：榨精量达标（→ 爆发模式）

  ### 状态 3：爆发模式（光明玷污）

  - **触发条件**：榨精量达标
  - **执行流程**：
    - 圣光全面反噬
    - 极致快感
    - 战炎彻底被榨
    - 暮黎被榨精画面同步
  - **终止条件**：天火指令（→ 恢复或循环）

  ### 状态 4：恢复模式（圣光恢复）

  - **触发条件**：榨精完成
  - **执行流程**：
    - 战炎短暂休息
    - 圣光能量恢复
    - 准备下一轮献祭
  - **终止条件**：圣光储量恢复（→ 循环至预缚模式）

  ### 联动机制

  - **自缚-反噬闭环**：战炎主动献祭 → 圣光反噬 → 极致快感
  - **暮黎-榨精闭环**：暮黎求助 → 战炎献祭 → 战炎被榨 → 暮黎解放
  - **光明-玷污闭环**：圣光能量被榨取 → "光明被玷污"心理打击
  - **主动-被动闭环**：战炎主动献祭却获得被动榨取的快感

  - "为救弟弟"的心理扭曲
