  ## 概述
  - 蓝星上人类因魔兽侵扰而龟缩聚居的"五大区"统称，是帝王战队赖以活动与补给的核心地缘框架。
  - 五大区分别命名为**神耀城**（中央）/ **铁脊城**（北）/ **凌海城**（东）/ **黑渊城**（西）/ **沐禾城**（南）。
  - 联邦、审判会、奥义学院等机构均围绕五大区铺开；魔神会在各区设有对应实验室与暗桩。

  ## 神耀城 / 第一区
  - **方位**：中央行政带
  - **风貌**：行政中枢与世家汇聚地，气候温和，建筑以白金色调为主，是蓝星权力与世家文化的中心；烈阳世家总部位于此。
  - **核心设施**：
    - **烈阳世家总部**：1 区烈阳世家是蓝星顶级世家之一，准星为下任家主继承人、夜鸦战队队长。
    - **审判会总部**：联邦军部直属力量，统辖五大区军警与对魔神会作战。
    - **联邦议事塔**：中央安全区行政中枢，统筹跨区事务。
  - **关键 NPC**：准星、列阳世家家主、皇后（SS 级主教，魔神会第一区负责人）
  - **标志威胁**：
    - 皇后：黑白权杖+时空留影魔法，捕获准星的执行者
    - 暗狮：魔神会第一区情报贩子，蛮力型
    - 夜鸦战队：准星所率小英雄职业小队（准星瞒着父母从事）
  - **小说锚点**：准星被皇后击败后，父母作为人质；皇后剥离出 4 个留影分身（一年只能用一次）

  ## 铁脊城 / 第二区
  - **方位**：北部重工业带
  - **风貌**：以重工业与机械化制造闻名，气候偏冷，城市被巨型工厂、束缚流水线、能量核心环绕；纪律森严，效率至上。
  - **核心设施**：
    - **束缚工厂**：机械化流水线、标准化束缚器具的生产基地
    - **能量核心**：联邦能源枢纽之一
    - **机械回收场**：废弃机械、合成系机械部件的交易与回收点
  - **关键 NPC**：赫菲斯托斯（白发老头，杯酒啃面包的疯狂科学家）、奥西里斯（机械创神主导者）
  - **标志威胁**：
    - 赫菲斯托斯实验室：负责研发"幽灵一号"少年英雄改造体
    - 幽灵一号：战力可达 S 级的少年英雄改造体
    - 黑胶铠甲、机械触手、流水线拘束器具

  ## 凌海城 / 第三区
  - **方位**：东部沿海商业带
  - **风貌**：联邦最大港口与贸易枢纽，气候湿润、海风强劲，城市被高耸的滨海建筑、闪烁的霓虹灯、繁忙的港口起重机点缀；开放、多元、商业理性与奢侈消费并存。
  - **核心设施**：
    - **奥义学院（与凌海城共址）**：审判会下辖最高学府，培养觉醒少年英雄
    - **C 市贫民窟**：朦胧清晨阳光洒落到第三区贫民窟的大街小巷，是文斌出身地
    - **中心大剧院**：魔神会主动投放"招 13—16 岁优质少年拍电影"小广告的诱饵地点
    - **凌海港口**：联邦最大贸易港口，海洋系魔物渗透风险最高
    - **滨海庄园 / 海滨路 78 号**：黑市总部，暗中交易被捕获的小英雄与高品质精能
  - **关键 NPC**：文斌（第三区贫民窟出身的孤儿）、战炎、暮黎、洛基（魔神会第三区实验室负责人）
  - **标志威胁**：
    - 洛基：化妆舞会主导者+黑科技展示（演讲台展示神话礼盒·螺旋背插等）
    - 中心大剧院狩猎：路西法巡回狩猎队狩猎战炎/暮黎
    - 奥西里斯第三区分部：主导文斌机械创神改造
  - **小说锚点**：文斌被卖到实验研究所成为活体实验材料、在人机改造过程中被问天救下；战炎捡到中心大剧院小广告后被狩猎；凯风在 C 市公园被公开调教为"肉便器"

  ## 黑渊城 / 第四区
  - **方位**：西部矿区法外带
  - **风貌**：边疆法外之地，气候干燥、多风沙，地表被废弃矿洞、深渊裂缝、古代遗迹覆盖；法律约束薄弱，魔神会活动密集。
  - **核心设施**：
    - **废弃矿洞**：与深渊裂缝相连，是原始寄生、共生型魔物的栖息地
    - **古代遗迹**：上古年间留下的时空裂缝与天材地宝
    - **黑渊地下实验区**：奥西里斯机械创神的核心实验场（需 SSS 级北地修罗虎血液、SSS 级传说物品机械星辉）
  - **关键 NPC**：奥西里斯（机械创神主导者）、休、盖布（奥西里斯副手）
  - **标志威胁**：
    - 奥西里斯实验室：负责 SSS 级机械创神改造（文斌改造在此完成）
    - 蠕动鞘管：包裹/束缚型魔物工具，模拟胃部蠕动将目标卷入矿洞
    - 寄生探针：侵入/内部型魔物工具
    - 红衣主教"第四区域红衣都出动好几位"代价出动
  - **小说锚点**：文斌被送往奥西里斯实验室进行机械创神改造，代价为第四区域红衣都出动好几位

  ## 沐禾城 / 第五区
  - **方位**：南部农业森林带
  - **风貌**：古老森林之地，气候温润多雨，被千年古木、温室、采石场、神秘遗迹覆盖；表面宁静、深处暗藏魔神会中下游据点。
  - **核心设施**：
    - **温室**：种植各类天材地宝，部分被魔神会控制
    - **沐禾古林**：原始魔物栖息地，蕴含自然共生型威胁
    - **沐禾采石场**：与古代遗迹交错，是秘境管理局登记的二级限制进入区
  - **关键 NPC**：魔神会中下游据点
  - **标志威胁**：
    - 自然共生型魔物：精络食精兽、生态拟态者（魔神会驯化）
    - 堕落教团：自然崇拜为掩护的魔神会分支
    - 盗猎者胶液球：内部侵入型魔物工具
  - **小说锚点**：烈阳世家所在的"第五区"地理交错；魔神会中下游据点隐于古林深处

  ## 关键事件
  - **第三区是帝王战队的主战场**：文斌与阚泽均出身边缘区；问天、千瑞、阚泽、诗华、凯风、嘉陵、空耀、子夜均与凌海城有关
  - **准星被捕获**：第一区烈阳世家附近被皇后击败并捕获
  - **凯风/战炎/暮黎被狩猎**：在凌海城 C 市公园/中心大剧院被公开调教或狩猎
  - **文斌改造**：在黑渊城奥西里斯实验室完成机械创神改造
  - **奥义学院前期培训**：问天/千瑞/阚泽等人在凌海城奥义学院培训期间结识并缔结羁绊

  ## 关键势力
  - **审判会**：联邦体系下军部直属力量，统辖安全区军警与对魔神会作战
  - **猎人协会**：注册猎人系统，阚泽、陶然等 S 级小英雄均为注册猎人
  - **奥义学院**：审判会下辖的最高学府
  - **暗桩与渗透**：魔神会在各区均设有化妆舞会、暗影交易所等渗透点
