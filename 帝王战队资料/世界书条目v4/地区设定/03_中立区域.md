  ## 概述
  - 安全区与魔神会控制区交错的灰色地带，以第三区 C 市贫民窟与黑市为典型代表。
  - 联邦、审判会、猎人协会的管辖力弱，魔神会暗桩活跃，是底层交易与小英雄诱饵的温床。

  ## 详细设定
  ### 第三区 C 市贫民窟
  - 朦胧清晨阳光洒落到第三区贫民窟的大街小巷，文斌在此处简陋民居中成长。
  - 贫民窟是文斌目睹魔兽攻城血腥屠戮、经历亲人生离死别的地方，"能活下去"是他当时的唯一信念。
  - 战炎、暮黎也曾出身边缘平民窟，常在公园、绿茵场、棒球场等公共空间活动。
  - 凯风曾在公园被人发现为"极品小英雄肉便器"——粉嫩小穴紧致、主动招待客人；知情人士神秘消失。

  ### 黑市与小广告
  - 黑市有"取自少年的精华"出售传闻；高品质少年英雄精液千金难求。
  - 中心大剧院（第三区）是魔神会主动投放"招 13—16 岁优质少年拍电影"小广告的诱饵地点。
  - 商业区有"夜色俱乐部"等高端社交场所，是鼹鼠、洛基等暗桩常驻地。
  - 黑市可买到天火伪装过的少年英雄精华（神魔号分析后失去基因活性，已伪装销售）。

  ### 势力交叠
  - 审判会、猎人协会在 C 市有驻地，监控小英雄失踪案。
  - 魔神会各区暗桩均可在 C 市活动，路西法巡回狩猎队即在此地活跃。
  - 联邦报纸的"今日头条"经常以最大版面报道小英雄堕落事件。

  ### 关键人物
  - 文斌：第三区贫民窟出身的孤儿，最终成为帝王战队机械天才。
  - 战炎、暮黎：出身平民窟的篮球骁将 / 棒球王牌，深夜成为魔神会的"胯间性奴"。
  - 凯风（时空凯风）：在 C 市公园被人目击"被公开调教为肉便器"，后被天火彻底收服。
  - 鼹鼠：魔神会在 C 市的暗桩头目。
  - 战炎、暮黎身边的"夜行者"小英雄群体。

  ## 关键事件
  - 文斌被卖到实验研究所成为活体实验材料，在人机改造过程中被问天救下。
  - 战炎捡到第三区中心大剧院的小广告，血脉喷张去拍宣传电影——被路西法的巡回狩猎队盯上。
  - 凯风在公园被公开调教后，知情人士与"目击者"在第二天中午全部神秘消失。
  - 陶然作为小英雄在 C 市被天火、文斌联手击败并调教。
