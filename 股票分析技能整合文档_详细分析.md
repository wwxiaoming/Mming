# 股票分析技能整合文档 · 详细分析（v2.0）

> 本文是 `/workspace/股票分析技能整合文档.md` 的逐技能深度解读，覆盖核心价值、关键能力、典型使用、优势局限与集成评价，并对整体架构给出改进建议与角色适配方案。
>
> **v2.0 变更**：基于 `/workspace/USAGE.md` 新增 10 个 skill，把架构从 4 层扩展为 5 层（L0 战术与全球宏观 + L1 资讯 + L2 选股 + L3 分析 + L4 产出），总技能数 14 → 24。

---

## 总体评估

整合文档对 TRAE 内 **24 个股票/金融相关技能** 做了体系化整合，构建了 **「L0 战术与全球宏观 → L1 资讯 → L2 选股 → L3 分析 → L4 产出」** 五层漏斗式工作流。优点是覆盖度全、跨市场衔接清晰、决策树实用；下面按层级逐个技能做深度拆解。

---

## 零、L0 战术与全球宏观层（v2 新增 · 6 个技能）

> 这是 v2.0 全新引入的最底层，定位为「跨市场战术 + 全球宏观」前置信号层。L0 给出大盘/板块/广度/见顶/资金流/大宗的量化锚点，是判断「今天值不值得做、风险预算给多少」的第一道闸门。L0 不直接产个股，但能对 L1-L4 的所有判断加上宏观滤镜。

### 0.1 `macro-rates-monitor` — 宏观 + 利率仪表盘

| 维度 | 分析 |
|------|------|
| **核心价值** | 用 LSEG MCP 聚合 GDP / CPI / PMI / 国债曲线 / 通胀盈亏平衡 / 掉期利差，给出当前经济周期位置 + 金融条件松紧 |
| **数据时效** | 月度宏观指标为主，季度 GDP；利率曲线实时 |
| **关键能力** | 4 段输出（Macro Summary / Yield Curve / Real Rate / Swap Spread）+ 综合判断 |
| **典型使用** | 工作流前预检：当前是「早周期/中周期/晚周期/衰退」？金融条件宽松还是紧缩？ |
| **局限性** | 强依赖 LSEG MCP 配置，无 MCP 时无法运行；不直接覆盖 A 股 |
| **集成评价** | ⭐⭐⭐⭐ 跨市场宏观的「总开关」 |

### 0.2 `sector-analyst` — 板块轮动 + 周期定位

| 维度 | 分析 |
|------|------|
| **核心价值** | 用 TraderMonty 公开 CSV 计算 11 个 GICS 板块的「站上 50/200 日均线比率」，定位当前周期阶段（早/中/晚/衰退） |
| **数据时效** | 日频 CSV，可每日刷新 |
| **关键能力** | 5 步工作流：CSV 抓取 → 周期评估 → 当前情况 → 情景发展（2-4 个 + 概率）→ 6 段报告 |
| **典型使用** | 工作流前预检：当前是周期股领涨还是防御股领涨？是否已经处于「晚周期」？ |
| **局限性** | 板块分类偏 GICS 标准，A 股板块映射需手工对齐；情景概率带主观判断 |
| **集成评价** | ⭐⭐⭐⭐ 板块层战术的「显微镜」 |

### 0.3 `market-breadth-analyzer` — 市场广度量化

| 维度 | 分析 |
|------|------|
| **核心价值** | 用 6 维度打分系统量化市场广度健康度（0-100），5 个分区（Strong/Healthy/Neutral/Weakening/Critical） |
| **数据时效** | 日频 GitHub Pages CSV |
| **关键能力** | 自动权重重分配（缺失数据按比例补）、历史百分位、跨运行持久化（最多 20 条） |
| **典型使用** | 与 market-top-detector 配合使用，作为大盘健康度的「温度计」 |
| **局限性** | 数据 > 5 天会发新鲜度警告；只覆盖 S&P 500 标的，跨市场需配合 L1 |
| **集成评价** | ⭐⭐⭐⭐ 大盘广度的「心电图」 |

### 0.4 `market-top-detector` — 见顶概率 0-100 分

| 维度 | 分析 |
|------|------|
| **核心价值** | 战术 2-8 周择时信号，预测 10-20% 回调概率，融合 O'Neil Distribution Day + Minervini 龙头股恶化 + Monty 防御板块轮动 |
| **数据时效** | FMP API 实时 + WebSearch 增量 |
| **关键能力** | 6 维度打分（Distribution Day / 龙头股 / 防御轮动 / 广度背离 / 指数形态 / 情绪 + 投机），5 个风险区（Green/Yellow/Orange/Red/Critical） |
| **典型使用** | 大盘仓位管理：见顶概率 ≥ 60 → 减仓 + 紧止损；≤ 20 → 满仓 + 动量 |
| **局限性** | 数据陈旧 > 3 天会严重降低质量；FMP 免费版 33 次/运行；不覆盖 A 股 |
| **集成评价** | ⭐⭐⭐⭐⭐ 2-8 周战术择时的「王牌」 |

### 0.5 `institutional-flow-tracker` — 13F 机构资金流

| 维度 | 分析 |
|------|------|
| **核心价值** | 通过 SEC 13F 季度申报跟踪对冲基金 / 共同基金 / 养老金的 smart money 累积/派发 |
| **数据时效** | 季度数据，滞后 45 天申报 |
| **关键能力** | 5 步工作流：扫描异动 → 个股深度 → 跟踪特定投资人 → 信号解读（5 档）→ 组合应用 |
| **典型使用** | 美股标杆标的资金流（NVDA / TSLA / AAPL），作为 A 股相关产业链的对标参考 |
| **局限性** | 45 天报告延迟 → 确认指标非领先；只覆盖管理规模 > $100M；只报多头；A 股映射有限 |
| **集成评价** | ⭐⭐⭐⭐ smart money 跟踪的「长矛」 |

### 0.6 `commodities-quote` — 大宗商品实时报价

| 维度 | 分析 |
|------|------|
| **核心价值** | 实时查询贵金属 / 能源 / 农产品 / 基本金属的当前价、涨跌、区间、均线、量价 |
| **数据时效** | Octagon MCP 实时报价 |
| **关键能力** | 10 字段指标（日内/52 周位置、50/200 均线）+ 5 种均线形态 + 4 种量价组合 + 商品基本面驱动 |
| **典型使用** | 资源股背景：黄金 → 黄金股；原油 → 三桶油 + 油服；铜 → 铜矿 + PCB 链 |
| **局限性** | 强依赖 Octagon MCP 配置；不直接覆盖个股；不替代产业链分析 |
| **集成评价** | ⭐⭐⭐⭐ 资源股环境的「标尺」 |

## 一、L1 资讯层（信息输入端）

### 1. `last30days` — 舆情与情绪触发器

| 维度 | 分析 |
|------|------|
| **核心价值** | 抓取 Reddit、X、YouTube、TikTok、HN、Polymarket、GitHub 等海外平台近 30 天的真实讨论，量化「热度」与「共识度」 |
| **数据时效** | 30 天滚动窗口，刚好覆盖一个完整的中期题材发酵周期 |
| **关键能力** | 多平台并行抓取 + 互动量（engagement）排序，能区分"小众深度讨论"与"散户情绪噪音" |
| **典型使用** | 工作流 D 启动器、判断海外催化剂是否传导到 A 股（如英伟达 GPU → A 股光模块、PCB 链） |
| **局限性** | 对中文圈覆盖弱，需配合 A 股个股新闻（a-stock-data 巨潮/东财）使用 |
| **集成评价** | ⭐⭐⭐⭐⭐ 唯一的「外部信号源」，不可替代 |

### 1.5 `akshare` — 中文金融数据快速通道（v2 新增）

| 维度 | 分析 |
|------|------|
| **核心价值** | 开箱即用的中文金融数据 Python 库，覆盖 A 股 / 港股 / 美股 / 期货 / 基金的实时与历史数据，叠加 GDP / CPI / PMI 宏观经济指标 |
| **数据时效** | 实时行情 + 历史日/周/月 K 线 + 财务三表 + 宏观月度数据 |
| **关键能力** | 全 pandas DataFrame 返回、零 API key、即装即用、支持前/后复权与多周期 |
| **典型使用** | 工作流 A/B 的中文数据通道（a-stock-data 不可用时的备选）；宏观背景（PMI / CPI）查询；港股美股联动标的的数据补充 |
| **优势** | 零门槛、覆盖全市场（含港美股）、社区活跃、文档齐全 |
| **局限** | 自带缓存弱，频繁请求易被封 IP；TTM 数据滞后；不替代深度估值（用 company-valuation / longbridge-research） |
| **集成评价** | ⭐⭐⭐⭐ 中文金融数据的「瑞士军刀」,a-stock-data 的最佳互补 |

## 三、L3 分析层（深度尽调端）

### 8. `a-stock-data` — A 股全栈数据工具包（七层数据源）

| 维度 | 分析 |
|------|------|
| **核心价值** | **整个整合文档中数据最厚的一个技能**，自包含零外部依赖 |
| **七层数据源** | ① 行情（mootdx+腾讯+百度K线）② 研报（东财+同花顺+iwencai）③ 信号（同花顺热点+北向+龙虎榜+解禁+行业）④ 资金面（融资融券+大宗+股东户数+分红+120日分钟级资金流）⑤ 新闻（东财个股+全球）⑥ 基础数据（mootdx财务/F10+东财+新浪三表）⑦ 公告（巨潮） |
| **工程亮点** | 内置东财限流防封、mootdx/腾讯不封 IP 兜底 |
| **典型使用** | 工作流 A/C/D 的数据底座；任何需要"原始数据 → 结构化结论"的场景 |
| **优势** | 覆盖度碾压同类、自带反爬策略、开箱即用 |
| **局限** | 调用代码量大，需根据场景挑层使用 |
| **集成评价** | ⭐⭐⭐⭐⭐ **整合文档中最核心的"数据中台"** |

| **数据时效** | 实时机构评级 + 季度 13F + 实时内部人交易 |

### 12.7 `company-valuation` — DCF + 相对估值 + SOTP 三方法（v2 新增）

| 维度 | 分析 |

| **数据时效** | yfinance 实时 + FMP CLI 可选 |
| **关键能力** | 4 步运行流程（环境检测 → 方法选择 → 三方法计算 → 9 段研报）+ 5×5 敏感性表 + Bull/Base/Bear 情景 |
| **典型使用** | 工作流 A/D 的估值锚点；判断个股"贵不贵"+"该值多少" |
| **优势** | 默认参数完整（用户不指定时自动用）、9 段结构稳定、敏感性比点估计重要 |
| **局限** | yfinance 数据非官方、TTM 滞后；DCF 是 garbage-in/garbage-out；非投资建议 |
| **集成评价** | ⭐⭐⭐⭐ 深度估值的「重武器」 |

## 五、整体架构评价

### 5.1 优点

1. **分层合理**：v2.0 扩展为 5 层（L0 战术与全球宏观 → L1 资讯 → L2 选股 → L3 分析 → L4 产出），天然对应"宏观预检 → 信号输入 → 候选生成 → 深度验证 → 成品输出"的投研漏斗。
2. **方法论覆盖完整**：既有量化（a-share-screener、sector-analyst、market-breadth-analyzer、market-top-detector）又有价值（longbridge-value-investing、china-stock-analysis、longbridge-research、company-valuation）还有产业链（serenity）还有情绪（last30days）还有宏观（macro-rates-monitor、commodities-quote）还有资金流（institutional-flow-tracker、akshare）。
3. **跨市场衔接清晰**：L0 层为跨市场战术提供量化锚点（宏观/广度/见顶/资金流/大宗），L1/L3 层通过 longbridge / akshare 补港美股数据。
4. **衔接明确**：文档第七节"技能联动速查表"给出了 8 条最常用衔接路径，可直接当 SOP 用。
5. **风险提示到位**：第八节强调"仅供研究支持，不构成交易建议"，且把 A 股特性（T+1、涨跌幅）内置到 a-share-analysis。
6. **L0 战术层为跨市场择时提供量化锚点**：见顶概率 0-100 分、广度 0-100 分、13F 资金流可作"风险预算"基线，避免主观判断。

### 5.2 可改进点

1. **缺少「复盘层」**：投资决策做完后，没有专门的「业绩归因/错误分析」技能，建议在工作流末尾增加 `stock-watcher` 表现回看。
2. **跨市场协同弱（v2 部分解决）**：v2.0 引入 L0 层 + longbridge/akshare 后，跨市场路径已成型，但「L0 战术信号 → A 股个股」的传导仍需手工映射，建议未来提供映射模板。
3. **风险/合规层缺失**：没有专门的风控/仓位管理技能，建议补充一个简单的组合管理工具（基于 market-top-detector 的风险预算表）。
4. **时序版本管理**：v2.0 已建立 changelog 机制（详见末尾版本行），新增技能时同步更新。
5. **v2 新增**「L0 战术信号与 A 股个股联动需在文档中显式说明」：当前 0.x 章节（macro-rates/sector-analyst 等）的"典型使用"段落已点出对标关系，但未形成统一映射表，建议下一版补充「A 股产业链 × 美股标杆」对照表。

### 5.3 适配建议（按用户角色）

## 六、总结

v2.0 整合文档构建的 **24 个技能** + **5 层架构** + **7 条核心工作流** + **10 套选股策略矩阵**，本质上是把 TRAE 打造成了一个**「多市场、多策略、可组合、有方法论支撑」**的投研操作系统。**L0 层是宏观与战术预检、`a-stock-data` + `akshare` 是双数据中台、`a-share-analysis` 是 A 股适配层、`serenity` 系列是独门心法、`longbridge` 系列是卖方研究框架、`consulting-analysis/investment-memo` 是交付件**——这 6 类构成核心，其余 18 个技能围绕它们做补强。按工作流模板调用即可覆盖 90% 的 A 股 + 70% 的跨市场投研需求。
