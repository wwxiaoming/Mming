## 总体评估

整合文档对 TRAE 内 **24 个股票/金融相关技能** 做了体系化整合，构建了 **「L0 战术与全球宏观 → L1 资讯 → L2 选股 → L3 分析 → L4 产出」** 五层漏斗式工作流。优点是覆盖度全、跨市场衔接清晰、决策树实用；下面按层级逐个技能做深度拆解。

## 零、L0 战术与 A 股宏观层（v2.1 调整为 A 股原生 · 6 个技能）

> 这是 v2.0 引入、v2.1 调整为 A 股原生工具的最底层，定位为「A 股战术 + 中国宏观」前置信号层。L0 给出大盘/板块/广度/见顶/资金流/大宗的量化锚点，是判断「今天值不值得做、风险预算给多少」的第一道闸门。L0 不直接产个股，但能对 L1-L4 的所有判断加上宏观滤镜。v2.1 之前为美股向工具（见文末附记 A1），现改用 `ASHARE_README.md` 6 个 A 股原生 skill 替代。

### 0.1 `china-macro-analyst` — 中国宏观与货币政策（v2.1 替换 macro-rates-monitor）

| 维度 | 分析 |
|------|------|
| **核心价值** | 跟踪 GDP / CPI / PPI / PMI / 社融 / M2 / 十年期国债收益率 / 央行公开市场操作，给出 A 股所处宏观周期 + 货币政策松紧 |
| **数据时效** | 月度宏观指标为主，季度 GDP；LPR / 央行操作周度 |
| **关键能力** | 评估货币政策与财政政策对 A 股和港股市场的影响，解读政策信号与利率敏感型板块（金融/地产/公用事业） |
| **典型使用** | 工作流前预检：当前是宽货币/紧货币/松紧平衡？社融是否回升？央行是否降准降息？ |
| **局限性** | 不覆盖美债收益率与美元流动性；港美股联动需配合 longbridge-* |
| **集成评价** | ⭐⭐⭐⭐⭐ A 股宏观的「总开关」 |

### 0.2 `sector-rotation-detector` — A 股板块轮动（v2.1 替换 sector-analyst）

| 维度 | 分析 |
|------|------|
| **核心价值** | 跟踪申万/中证/中信 31 个一级行业 + 134 个二级行业的相对强弱与资金流向，识别周期 vs 防御轮动 |
| **数据时效** | 日频，可每日刷新 |
| **关键能力** | 行业 ROE + 毛利率 + 估值分位 + 北向资金持仓变化；周期/防御判断 + 轮动策略建议 |
| **典型使用** | 工作流前预检：当前是周期股领涨还是防御股领涨？主线行业是否扩散？ |
| **局限性** | 申万/中证分类与 GICS 不可直接对照；情景判断带主观性 |
| **集成评价** | ⭐⭐⭐⭐⭐ A 股板块层战术的「显微镜」 |

### 0.3 `limit-up-pool-analyzer` — 涨停池与连板梯队（v2.1 替换 market-breadth-analyzer）

| 维度 | 分析 |
|------|------|
| **核心价值** | 跟踪每日涨停家数、连板高度（一板/二板/三板/高位板）、炸板率、封板资金、涨停题材归因；输出 A 股广度 0-100 分 |
| **数据时效** | 盘后 T+0，更新实时 |
| **关键能力** | 5 项广度评分：涨跌停比 + 连板梯队 + 炸板率 + 封板率 + 新高新低比 → 0-100 综合分 |
| **典型使用** | 大盘情绪温度：≥ 80 → 情绪高位（警惕见顶）；40-60 → 正常；≤ 20 → 情绪冰点（关注反转） |
| **局限性** | 仅覆盖主板涨停；不覆盖北交所/科创板的 ±20% 涨跌幅；不预测持续性 |
| **集成评价** | ⭐⭐⭐⭐⭐ A 股情绪广度的「心电图」 |

### 0.4 `equity-pledge-risk-monitor` — 股权质押见顶预警（v2.1 替换 market-top-detector）

| 维度 | 分析 |
|------|------|
| **核心价值** | 跟踪全市场质押比例、控股股东质押率、平仓线/警戒线距离、质押市值与日均成交额比；识别高质押率个股（>50%）、质押股触及平仓线预警；输出 0-100 见顶/踩踏风险分 |
| **数据时效** | 日频 + 季度股东数据 |
| **关键能力** | 参考 2018 年 A 股闪崩与股权质押危机历史经验；2018 风险特征对标 |
| **典型使用** | 大盘风险预算：风险分 ≥ 60 → 减仓 + 紧止损；< 20 → 正常仓位 |
| **局限性** | 不预测大盘走势；股权质押信息有滞后；不适用于纯题材炒作无实质产业逻辑的小票 |
| **集成评价** | ⭐⭐⭐⭐ A 股见顶预警的「内生信号」 |

### 0.5 `fund-screener` — 公募基金筛选与 smart money（v2.1 替换 institutional-flow-tracker）

## 一、L1 资讯层（信息输入端）

### 1. `last30days` — 舆情与情绪触发器

| 维度 | 分析 |
|------|------|
| **核心价值** | 抓取 Reddit、X、YouTube、TikTok、HN、Polymarket、GitHub 等海外平台近 30 天的真实讨论，量化「热度」与「共识度」 |
| **数据时效** | 30 天滚动窗口，刚好覆盖一个完整的中期题材发酵周期 |
| **关键能力** | 多平台并行抓取 + 互动量（engagement）排序，能区分"小众深度讨论"与"散户情绪噪音" |
| **典型使用** | 工作流 D 启动器、判断海外催化剂是否传导到 A 股（如英伟达 GPU → A 股光模块、PCB 链） |
| **局限性** | 对中文圈覆盖弱，需配合 A 股个股新闻（a-stock-data 巨潮/东财）使用 |
| **集成评价** | ⭐⭐⭐⭐⭐ 唯一的「外部信号源」，不可替代 |

### 1.5 `akshare` — 中文金融数据快速通道（v2 新增）

| 维度 | 分析 |
|------|------|
| **核心价值** | 开箱即用的中文金融数据 Python 库，覆盖 A 股 / 港股 / 美股 / 期货 / 基金的实时与历史数据，叠加 GDP / CPI / PMI 宏观经济指标 |
| **数据时效** | 实时行情 + 历史日/周/月 K 线 + 财务三表 + 宏观月度数据 |
| **关键能力** | 全 pandas DataFrame 返回、零 API key、即装即用、支持前/后复权与多周期 |
| **典型使用** | 工作流 A/B 的中文数据通道（a-stock-data 不可用时的备选）；宏观背景（PMI / CPI）查询；港股美股联动标的的数据补充 |
| **优势** | 零门槛、覆盖全市场（含港美股）、社区活跃、文档齐全 |
| **局限** | 自带缓存弱，频繁请求易被封 IP；TTM 数据滞后；不替代深度估值（用 company-valuation / longbridge-research） |
| **集成评价** | ⭐⭐⭐⭐ 中文金融数据的「瑞士军刀」,a-stock-data 的最佳互补 |

## 二、L2 选股层（候选池构建端）

### 2. `a-share-screener` — A 股多策略批量筛选器

| 维度 | 分析 |

### 4. `longbridge-value-investing` — 格雷厄姆/巴菲特方法论

| 维度 | 分析 |
|------|------|
| **核心价值** | 唯一同时覆盖 Graham（NCAV 烟蒂）+ Buffett（经济护城河）双方法论的技能 |

| **局限** | 对成长股、题材股不友好，需配合 china-stock-analysis |
| **集成评价** | ⭐⭐⭐⭐⭐ 价值投资者的"宪法" |

## 三、L3 分析层（深度尽调端）

### 8. `a-stock-data` — A 股全栈数据工具包（七层数据源）

| 维度 | 分析 |
|------|------|
| **核心价值** | **整个整合文档中数据最厚的一个技能**，自包含零外部依赖 |
| **七层数据源** | ① 行情（mootdx+腾讯+百度K线）② 研报（东财+同花顺+iwencai）③ 信号（同花顺热点+北向+龙虎榜+解禁+行业）④ 资金面（融资融券+大宗+股东户数+分红+120日分钟级资金流）⑤ 新闻（东财个股+全球）⑥ 基础数据（mootdx财务/F10+东财+新浪三表）⑦ 公告（巨潮） |
| **工程亮点** | 内置东财限流防封、mootdx/腾讯不封 IP 兜底 |
| **典型使用** | 工作流 A/C/D 的数据底座；任何需要"原始数据 → 结构化结论"的场景 |
| **优势** | 覆盖度碾压同类、自带反爬策略、开箱即用 |
| **局限** | 调用代码量大，需根据场景挑层使用 |
| **集成评价** | ⭐⭐⭐⭐⭐ **整合文档中最核心的"数据中台"** |

### 9. `a-share-analysis` — A 股综合分析（v2.1 强化为 company-valuation 的 A 股替代）

| 维度 | 分析 |

| **关键能力** | A 股特性（T+1、涨跌幅限制、北向资金、停复牌、申万行业）+ 政策影响评估 + 三方法估值（DCF / 相对 / SOTP）+ 5×5 敏感性表 |

| **集成评价** | ⭐⭐⭐⭐⭐ A 股个股分析的"瑞士军刀" + A 股估值的"合规重武器" |

## 五、整体架构评价

### 5.1 优点

1. **分层合理（v2.1 A 股化）**：v2.1 把 5 层架构的 L0 层从「全球宏观」调整为「A 股宏观 + A 股战术」，主流程全面 A 股化，天然对应"A 股宏观预检 → A 股信号输入 → A 股候选生成 → 深度验证 → 成品输出"的 A 股投研漏斗。
2. **方法论覆盖完整（v2.1 A 股化）**：既有 A 股量化（a-share-screener、sector-rotation-detector、limit-up-pool-analyzer、equity-pledge-risk-monitor）又有 A 股价值（longbridge-value-investing、china-stock-analysis、a-share-analysis 强化估值）还有 A 股产业链（serenity）还有 A 股情绪（last30days 海外映射 + 涨停池）还有 A 股宏观（china-macro-analyst）还有 A 股资金流（fund-screener、akshare）。
3. **A 股原生优先**：L0-L4 主流程的所有数据/方法论工具均为 A 股原生，与 ASHARE_README.md 6 个 A 股 skill 完全对齐。跨市场工具（longbridge / akshare / 6 个美股 skill）作为辅助信号源。
4. **衔接明确**：文档第七节"技能联动速查表"给出了 8 条最常用衔接路径，可直接当 SOP 用。
5. **风险提示到位**：第八节强调"仅供研究支持，不构成交易建议"，且把 A 股特性（T+1、涨跌幅、停复牌、北向）内置到 a-share-analysis。
6. **L0 战术层为 A 股战术择时提供量化锚点**：涨停池/连板梯队 0-100 分、股权质押见顶 0-100 分、公募 smart money 跟踪、申万板块轮动可作 A 股"风险预算"基线。

### 5.2 可改进点

1. **缺少「复盘层」**：投资决策做完后，没有专门的「业绩归因/错误分析」技能，建议在工作流末尾增加 `stock-watcher` 表现回看。
2. **跨市场协同弱（v2 部分解决）**：v2.0 引入 L0 层 + longbridge/akshare 后，跨市场路径已成型；v2.1 把 L0 A 股化后，「美股 → A 股」传导需通过 longbridge-earnings（业绩前瞻） / longbridge-research（13F 标杆）/ akshare（H 股数据）实现。
3. **风险/合规层缺失**：没有专门的风控/仓位管理技能，建议补充一个简单的组合管理工具（基于 equity-pledge-risk-monitor 的风险预算表）。
4. **时序版本管理**：v2.1 已建立完整 changelog 机制（详见末尾版本行），新增技能时同步更新。
5. **v2.1 解决**「L0 战术信号与 A 股个股联动」：v2.1 把 L0 改为 A 股原生工具后，信号与 A 股个股直接打通，无须手工映射。

## 六、总结

v2.1 整合文档构建的 **24 个技能**（A 股原生主流程 + 美股/港美股附记）+ **5 层架构**（L0 A 股化）+ **7 条核心工作流** + **10 套选股策略矩阵**，本质上是把 TRAE 打造成了一个**「A 股为主、跨市场可选、可组合、有方法论支撑」**的投研操作系统。**L0 层是 A 股宏观 + A 股战术预检、`a-stock-data` + `akshare` 是双数据中台、`a-share-analysis` 是 A 股适配层兼估值重武器、`serenity` 系列是独门心法、`fund-screener` 是 A 股 smart money、`limit-up-pool-analyzer` 是 A 股情绪广度、`sector-rotation-detector` 是 A 股板块轮动、`china-macro-analyst` 是 A 股宏观、`equity-pledge-risk-monitor` 是 A 股见顶预警、`consulting-analysis/investment-memo` 是交付件**——这 11 类构成核心，其余 13 个技能围绕它们做补强。按工作流模板调用即可覆盖 95% 的 A 股投研需求 + 60% 的跨市场协同需求。
