# 00 · research 目录使用说明书

> **目录性质**：WQ BRAIN 顾问问卷备考的**研究素材库**（共 6 份文件）
> **配套使用**：本目录的 5 份文件（01-05）按统一结构组织
> **使用流程**：AI 写思路 → 用户去 IMA 提问 → 用户回填 AI 改写

---

## 0. 30 秒看懂工作流

你看到的截图（DeepSeek 里搜"矩阵向量数据风格改写"）就是**这套工作流的标准用法**。

```
┌─────────────────────────────────────────────────────────┐
│  Step 1 · AI（我）先写思路                                │
│  在 research/ 的对应文件里，每道题都有"思路"小节            │
│  ——告诉你"应该讲什么、按什么结构讲"                        │
└────────────────────┬────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────┐
│  Step 2 · 你去 IMA 知识库 / DeepSeek 提问                  │
│  在 research/ 的对应文件里，每道题都有"AI 提问模板"         │
│  ——一段可直接复制粘贴的提示词                              │
└────────────────────┬────────────────────────────────────┘
                     ↓
┌─────────────────────────────────────────────────────────┐
│  Step 3 · 你把 IMA 资料回填，AI 帮你改写                   │
│  在 research/ 的对应文件里，每道题都有"回填改写区"          │
│  ——你把 IMA 的回答贴进来，我按"3 板斧"改写                 │
└─────────────────────────────────────────────────────────┘
```

**核心心态**：AI 是**翻译机**和**参考书**，不是**答题器**。你的理解 + 你的表达 = 最佳答案。

## 2. 推荐使用顺序

### 🟢 第一轮（30 分钟）：建立全局观
1. 读 [00-工作流总览.md](00-工作流总览.md)（你现在这一步）
2. 读 [05-乌龙教程答题方法论.md §1-§4](05-乌龙教程答题方法论.md) ——了解"AI 味红线词"和"改写 3 板斧"
3. 读 [04-community-experience.md §5](04-community-experience.md) ——了解反作弊机制

### 🟡 第二轮（每个题 20-30 分钟）：按题作答
对每道题（Q4-Q9），按以下顺序操作：

1. **打开 [01-wq-learn-concepts.md](01-wq-learn-concepts.md)** 找到对应章节 → 看"思路模板"
2. **复制"AI 提问模板"** → 粘贴到 DeepSeek / 豆包 / IMA
3. **看 IMA 回答** → 复制回 [03-ima-qa.md](03-ima-qa.md) 或 [05-乌龙教程答题方法论.md](05-乌龙教程答题方法论.md) 对应章节的"回填改写区"
4. **告诉我**"Q4 改写好了，请帮我检查 AI 味" → 我帮你扫一遍

### 🔴 第三轮（5-10 分钟）：交卷前自检
1. 读 [05-乌龙教程答题方法论.md §9 自检清单](05-乌龙教程答题方法论.md)
2. 一项一项过
3. 重点检查：**ID / 邮箱** 没填错

## 3. 5 分钟上手（Q4 完整示例）

### 3.1 Step 1：读思路

打开 [01-wq-learn-concepts.md §1 Q4](01-wq-learn-concepts.md)，你会看到：

> **思路模板**（Q4）：
> - 必讲点 1：二维 vs 三维的形状差异
> - 必讲点 2：每天每只股票 1 个数 vs N 个数
> - 必讲点 3：必须用 `vec_` 算子聚合才能用
> - 推荐结构：定义 → 举例 → 金融含义
> - 必避坑：不要答成"数学向量"（这里的"向量"是事件型数据）

### 3.2 Step 2：去 DeepSeek 提问

复制下面这段（这是 `01-wq-learn-concepts.md` 里"AI 提问模板"的真实内容）：

```
你是一个 BRAIN 平台顾问备考助手。
我刚收到问卷第 4 题：What is the difference between Matrix Data and Vector Data?
请用 2-3 句话回答，区分清楚两者的形状、典型例子、为什么需要 vec_ 算子。
要求：
- 不要用括号加注释
- 不要用"举个例子便于理解"开头
- 不要用"注意："作为开头
- 给我一个具体字段例子
```

把这段贴到 DeepSeek / 豆包 / IMA，AI 会回答一段话。

### 3.3 Step 3：回填改写

把 AI 的回答粘贴到 [03-ima-qa.md §2.1 Q4 的"回填改写区"](03-ima-qa.md)，告诉我：

> "Q4 帮我按 3 板斧改写：加我自己（anl4 经历）、换比喻（班级平均分）、调语序（先抛数字再讲定义）"

我就会把 AI 答案改写成**你自己风格**的版本。

### 3.4 Step 4：去平台填写

拿着改写后的答案，到 BRAIN 平台问卷页面粘贴、提交。

## 4. 三件套的具体位置

每道题（Q4-Q9）在主用方法论文件（01 / 03 / 05）里都有三件套：

| 三件套 | 在文件里叫什么 | 长什么样 |
|---|---|---|
| **思路模板** | `### 思路模板` | 必讲点 + 必避坑 + 推荐结构 + 200 字以内 |
| **AI 提问模板** | `### AI 提问模板` | 4 段式：角色 + 任务 + 输出格式 + 风格约束 |
| **回填改写区** | `### 回填与改写区` | 空白粘贴框 + 改写指令 + 必删 AI 味词清单 |

**避坑章节（04 的雷区、02 的 Workday/协议）不设三件套**——它们是"避坑"内容，不直接对应问卷作答。

## 6. 关键提醒

⚠️ **本目录所有内容**：
- ✅ 包含**思路模板**和**AI 提问模板**（可直接用）
- ❌ **不**包含"可直接照抄的答案"（违反 BRAIN 平台反作弊规则 + 你自己立的规矩）
- ✅ 用户必须通过"AI 提问 → 回填 → 改写"工作流自己组织答案

⚠️ **关于反作弊**：
- 平台用 4 类机制：文字相似度 + 语义相似度 + 答题时长 + IP/设备指纹
- **不**使用 "PCA + 哈希" 等细节（无官方/社区出处，请勿引用）

⚠️ **关于中性化定义**：
- **BRAIN 默认中性化 = 组内减均值**
- z-score 标准化是**可选**的进一步处理，**不**是中性化本身
- 01 / 03 / 04 三份文件已统一为此表述
