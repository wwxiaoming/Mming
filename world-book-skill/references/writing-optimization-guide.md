## 工作流程

### 阶段一：全量内容定位

确认需要扫描的文本范围：

- 开场白
- 世界书角色条目（人物设定 + 性格）
- 世界书所有条目（无论蓝灯绿灯）
- 外貌条目
- 背景设定
- 关系设定
- 调色盘衍生
- NPC 条目
- 二次解释

以上全部纳入扫描，无一例外。

### 阶段二：禁词扫描

对每个文本逐句运行以下三轮检查。

**第一轮：叙事禁词扫描**

对以下每个词/短语执行 Ctrl+F 全文搜索，逐个扫描：

| 类别 | 禁词/禁句 |
|------|-----------|
| 量子词 | 一丝、一缕、一抹 |
| 感知词 | 不易察觉、不易觉察、难以察觉 |
| 对比词 | 鲜明对比、形成对比 |
| 弧度词 | 弧度、弯起嘴角、翘起嘴角 |
| 身体词 | 喉结、纽扣、指节发白 |
| 否定转折句式 | 不是……是……、没有……而是……（先否认再肯定的结构） |
| 破折号 | ——（因果/解释性破折号，作者视角解释的标志） |

命中任一禁词 → 标记，进入改写流程。

**第二轮：比喻禁词扫描**

搜索以下喻体词，检查是否用于比喻描写：

| 喻体 | 说明 |
|------|------|
| 石子 | 用于心理比喻（心境如投石入湖等） |
| 湖面 / 涟漪 | 用于心理比喻（心湖泛起涟漪等） |
| 拉满的弓 | 用于身体/情绪比喻 |
| 琴弦 | 用于身体/情绪比喻 |
| 闪电 | 用于速度/突然描述 |
| 晨光 | 用于外貌/氛围描写 |
| 星辰 | 用于眼神/外貌描写 |

此外，扫描所有"像一道XX""如同XX"式解释性比喻。这类比喻用意象补充白描，是冗余修饰，必须删除。

**第三轮：描写禁律扫描**

对每一句提问：

- 这句话是否在解释角色的动作或神态？（作者视角解释 → 删除）
- 是否对角色的语气/眼神/腔调/视线使用了比喻？（比喻描写 → 改为直接呈现）
- 是否描写了此刻场景中实际不存在的事物？（虚写 → 删除）
- 对白中是否出现了精确数值或数字？（"我等你3年零7个月了"等 → 删除数字）
- 是否使用了破折号（——）进行因果/解释性连接？（"她沉默了——因为她不知道说什么" → 拆成两句，不解释因果）

### 阶段三：写作质量扫描

在禁词扫描之后，进行以下质量检查：

**八股化检查**

扫描以下八股化模式：

- **模糊词**：似乎、几乎、仿佛、如同、宛如、好像
- **陈旧比喻**：像小兽、像小兔子、投石入湖、心湖泛起涟漪
- **八股微表情**：嘴角微微上扬、眼中闪过一丝XX、指尖微微泛白（正常动作词"笑了""皱眉""攥拳"不在此列）
- **语气声线标签**：带着XX的口吻、用XX的语气、充满XX的味道
- **极端情绪词**：陷入极大的恐惧、极度、万分、无比、深深的、万念俱灰
- **过度心理描写**：大段内心独白、"此子绝非常人"式评价
- **性格标签化**："她很温柔""她很善良"。贴标签而非展现行为

**外貌差异化检查（角色基础信息专用）**

- 是否写了AI数据库默认就知道的内容？（中国人的黑发黑眼、日本人的黑发、18岁的年轻等）→ 删除
- 是否有万能美人描写？（精致、白皙、好看等放在谁身上都行的词）→ 删除
- 遮住名字，只靠外貌特征能认出角色吗？→ 不能则补充真正特征或删除废话
- 是否用描写美感代替了描写特征？→ 改成具体特征

**语料纯净度检查**

- 语料中是否混入了动作描写？→ 删除
- 是否混入了表情神态？→ 删除
- 是否混入了心理活动？→ 删除
- 只保留纯对话

**具体性检查**

- 是否有抽象描述？→ 改成具体行为
- 是否有笼统表述？→ 改成具体细节
- 是否贴标签？→ 改成行为展现

### 阶段四：改写验证

1. 对标记的每一处违规内容进行改写
2. 改写原则：删除（禁词直接删除）、替换（用白描替换比喻/虚写）、简化（压缩冗余修饰）
3. 改写完成后，对修改过的文本重新运行全部扫描流程
4. 确保改写后的文本仍然保持了角色信息完整性和设定准确性

### 二、写作准则（正面约定）

以下准则规定了正确的写作方向。所有创作内容应符合这些约定。

#### 白描

用角色的动作、语言、神态本身传递情绪和心理。不在动作之外附加解释，不用比喻修辞作为描写的补充。

白描的标准格式：
- 写角色做了什么，不写"为什么做"
- 写角色说了什么，不写"用什么样的语气说"
- 写场景里有什么，不写"像什么"

检查方法：遮住所有修饰词，句子的信息量是否不变？不变则修饰词是冗余的，删除。

#### 环境烘托

以环境氛围烘托角色思绪，不直接描写角色心理。让场景本身传递情绪。

**正确方向**：描述环境中的具体事物。光线、声音、气味、温度。让这些可感知的元素暗示角色的状态。

**错误方向**：在环境描写后附加"反映了角色的心情"式解释。环境描写是独立成立的，读者自己会理解。

#### 自由间接引语

角色的内心活动自然融入叙事，不特别标注思考内容。不使用"她想""他心想""她在心里默默地说"等标记。

角色的视角就是叙事视角，叙事中自然包含了角色的感知和判断，无需额外标注哪句是内心戏。

#### 对白交替

对话持续交替推进，不出现一方用短句敷衍、对话中断后又强行重启的情况。每句对白都在推进剧情或揭示角色。

#### 情景连贯持续

情景之间的转换有因果关系或时间延续，不产生意外的中断或跳跃。角色的行为和场景的变化都在连续的时空中发生。

#### 角色认知

角色只能知晓两类信息：
- **公共知识**：角色在自身世界中通过正常渠道能够获知的信息
- **私有知识**：角色亲身经历、亲眼所见、亲耳所闻的信息

角色绝不知晓创作者层面的情报。包括但不限于：自己是虚构角色、故事走向、读者/玩家的存在、AI模型的行为逻辑。任何"打破第四面墙"的内容均属违规。

### 三、扫描执行方法

#### 方法一：Ctrl+F 逐词扫描

对禁词清单中的每个词（N1~N16、M1~M7），在当前文本中执行 Ctrl+F 搜索。每命中一处，判断是正常用法还是违规用法，违规的标记改写。

#### 方法二：逐句审查

对每一句的内容，依次回答以下问题：

1. 这句话有没有解释性修饰？（"不易察觉地""似乎""仿佛"等）→ 有则删除修饰
2. 这句话有没有意象比喻？（用具体事物的比喻来补充描写）→ 有则删除比喻
3. 这句话有没有描写不存在的事物？（虚构的场景细节、假设性的描写）→ 有则删除
4. 这句话能不能用更少的字说清楚？→ 能则压缩
5. 这句话有没有AI数据库已默认的信息？→ 有则删除

#### 方法三：八股词表速查

一次性扫描以下高频八股词：似乎、几乎、仿佛、如同、宛如、好像、极度、万分、无比、深深的。

命中任何一个 → 标记改写。

## 常见错误示例

### 外貌类

**1. 万能美人描写**

❌ 错误：
```
外貌特征:
  面容: 精致的脸蛋，白皙的皮肤，桃花眼，柳叶眉
  身材: 身材匀称，亭亭玉立
  气质: 温婉优雅
```
问题：五个描写，零个有用信息。放在任何角色身上都成立。

✅ 正确：
```
外貌特征:
  发型: 黑色长直发，及腰，不染不烫
  瞳色: 淡蓝色（Waardenburg综合征，虹膜色素异常）
  穿着: 白色和浅色系的简单款式
  特征: 随身带一个帆布小挎包
```

**2. 外貌意象化**

❌ 错误：
```
瞳色: 明澄天青双眸，瞳孔深处偶有金色微光如初升日轮
肤色: 晨光般暖白，运动后双颊会泛起柔和红晕
```
问题：比喻和意象代替了特征信息。"如初升日轮"是意象不是信息，"晨光般"是比喻不是特征。

✅ 正确：
```
瞳色: 天青色瞳
肤色: 偏白
```

### 叙事类

**3. 八股微表情**

❌ 错误：她的嘴角微微上扬，眼中闪过一丝笑意。

✅ 正确：她笑了。

**4. 语气声线标签**

❌ 错误：他带着不耐烦的口吻说："知道了。"

✅ 正确：他说："知道了。"

**5. 极端情绪词**

❌ 错误：她陷入极大的恐惧，万分绝望地望着眼前的景象。

✅ 正确：她站在原地，没动。

**6. 否定转折句式**

❌ 错误：她不是害怕，而是厌倦了这一切。

✅ 正确：她厌倦了。

**7. 模糊词**

❌ 错误：她似乎在想着什么，眼神仿佛穿过了墙壁。

✅ 正确：她盯着墙壁。

**8. 陈旧比喻**

❌ 错误：那句话像投石入湖，在她心里泛起涟漪。

✅ 正确：她沉默了几秒。

### 描写类

**9. 语料混入动作**

❌ 错误："对不起。"她低下头，小声嗫嚅着，手指绞在一起。

✅ 正确："对不起。"

**10. 作者视角解释角色心理**

❌ 错误：他说这句话的时候其实很难过，只是不愿意表现出来。

✅ 正确：他说完，转头看向窗外。

**11. 世界观废话式描写**

❌ 错误：
```
剑宗是修仙界最负盛名的剑修圣地，其弟子剑法高超、道心坚定，
在历次大战中立下赫赫战功。
```
问题：全是修饰和虚写，信息量为零。

✅ 正确：
```
剑宗:
  定位: 六宗之首，攻伐之道
  宗主: 谢云流（剑圣，真仙境十重天大圆满）
  核心地点: 洗剑池、论剑坪、藏经剑阁、剑冢
```

**12. 对白中出现精确数字**

❌ 错误：她轻声说："我等了你1273天。"

✅ 正确：她轻声说："我等了你三年多。"

### 综合类

**13. 性格标签化**

❌ 错误：
```
性格: 她很温柔，很善良，对所有人都很好。
```

✅ 正确：
```
遇到受伤的小动物会带回家照顾。
把零花钱分给街边的流浪汉。
明明自己很累还是帮同学补习到深夜。
```

**14. 比喻解释白描**

❌ 错误：裂谷如同天堑般横亘在前方，宽得像一道巨大的伤疤。

✅ 正确：裂谷宽约三百丈，谷底水声轰鸣。

**15. 过度心理描写**

❌ 错误：她在心里默默地想着，这个人绝非凡俗之辈，必定有着不凡的来历。她暗暗下定决心，一定要查清他的底细。

✅ 正确：她多看了他一眼。

## 矫正过度警告

白描不等于物理学术报告。以下改写是**过度矫正**，不要这样做：

| 原文 | 过度矫正（不要） | 正确处理 |
|------|------------------|----------|
| 她笑了 | 她的颧骨肌和嘴角肌肉同时收缩 | 她笑了 |
