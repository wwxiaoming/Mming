# 一、世界观总览

## 1.1 世界背景——灵气复苏后的蓝星

自灵气复苏以来，蓝星空气中充斥着高浓度元素微粒。大量动物吸收元素后异化成形态各异的魔兽，人类被迫退守至唯一的安全区大陆。其余大陆与广阔海洋均沦为魔兽领地，安全区内各城市时常遭受魔兽袭击，魔兽攻城、屠村事件频发，人类文明在魔兽的阴影下艰难维系。

安全区划分为多个行政区，各区城市规模不一，防御等级不同。城市外围设有魔力屏障与巡逻防线，但仍无法完全阻挡高级魔兽的渗透与大规模兽潮。

安全区的东南方有一片荒芜的沙漠——**死寂沙漠**，一株参天古树在沙漠中央耸立，树枝上挂有暗红色果实。每到果实将要成熟之际就会散落点点荧光唤醒周遭的埋骨亡灵，拱卫在古树附近，在它周围还徘徊游荡着饥渴贪婪的各类魔兽，伺机争夺树上的果实。这就是被审判会划分为人类禁区之一的——死寂沙漠，而那颗古树就是天地灵物——**蒲罗树**。

## 1.2 人类职业体系

蓝星人类通过不同途径觉醒力量，形成三大主流职业：

- **异能者**：少数人能与空气中的魔力元素产生共鸣，根据共鸣元素种类分为火、冰、雷、风、土，光，暗、秩序、轮回、乾坤、时空、神魔、超能、圣灵、星辰、天道、幻等不同系别。异能强度与元素亲和度、身体容器容量直接相关，觉醒越早、越年轻的个体潜力越高。
- **机械师**：凭借强大大脑能力，利用科技手段对自身进行改造，植入机械义肢、神经接口、能量核心等装置，超越人类生理极限。改造程度越高，战斗力越强，但需持续维护能量供给。
- **御兽师**：能与魔兽签订契约，组成战斗伙伴。契约强度取决于双方精神契合度与魔力共振，可通过共同战斗提升默契，实现人兽合体或辅助作战。

部分特殊个体同时具备多重职业特征，或在觉醒过程中融合多种力量，潜力远超常人。

## 1.3 主要势力与机构

- **纠察委员会**（政府直属）：负责处理普通人与人之间的纠纷，调查常规违法犯罪事件，维护城市日常秩序。
- **审判会**（军部直属）：核心职能包括猎杀城市内潜伏魔兽、调查异能相关犯罪、打击魔神会等邪恶组织、开拓秘境遗迹、守护城市安定。审判会高度重视人才培养，下辖最高学府奥义学院。奥义学院招收各地中学完成基础教育且已觉醒能力的青少年，进行系统培养，为安全区输送抵抗魔兽的前线人才。同时吸收部分提前觉醒的少年组成特殊战队，在假期接受强化训练后下放至各城市中学，协助当地审判会执行日常任务，积累实战经验。优秀战队成员在未来进入奥义学院时可获得资源倾斜，并在协助城防期间接受官方宣传，成为公众认可的少年英雄。
- **猎人协会**（民间自发）：发布委托任务，包括狩猎指定魔兽、私人调查、护送、资源采集等。接受委托的猎人可自由组队，报酬根据任务难度与完成度结算，是许多觉醒者初期历练的重要渠道。

## 1.4 秘境、容器理论与元素能量机制

各地不时出现时空裂缝形成的秘境，内部孕育吸收魔力元素后诞生的天材地宝。这些宝物能显著提升异能强度，或帮助普通人类觉醒特殊能力。世家财阀与政府积极争夺秘境控制权。

核心科学理论为**容器理论**：人类身体本质上是一个巨大的元素能量容器。能力越强、年龄越小、身体机能越优秀的个体，容器容量越大，能吸纳并储存的元素微粒越多。

- 低级魔兽袭击人类主要为获取食物。
- 高级魔兽则瞄准人类的容器特性，吞噬人体可快速吸收大量天地元素。
- 人类之间无法直接吞噬对方体液或能量，但强者陨落后，在特定天地条件下，遗体可转化为纯粹的元素能量结晶，供后人吸收。因此，强者遗蜕是稀有资源，通常由其家族继承。
- 少年因容器容量最大，成为魔兽与部分邪恶势力优先猎食目标。黑市偶尔出现取自少年的"精华"交易，虽主流科学已证明普通少年基因原液无显著提升效果，但特殊少年的体液确实具备增强元素亲和、提升异能的罕见作用。这类特殊少年战斗能力优秀、元素亲和度更高，同时也是高级魔兽与魔神会的首要狩猎对象。

## 1.5 魔神会——邪恶组织

魔神会由各类失意者、心理扭曲者、被世家迫害者、变态社会人等组成，成员动机多为满足个人欲望，常在各区兴风作浪，煽动魔兽动乱。其内部等级体系清晰：

**使徒 → 神父 → 主教 → 枢机主教 → 红衣大主教 → 教宗 → 教皇**

魔神会核心能力包括利用高级魔兽追踪狩猎特殊少年，通过淫纹等手段进行精神与肉体改造，强化自身实力。魔神会实验室持续开发黑科技，如抑制魔力恢复的药剂、空间传送装置、时间操控道具、生物乳胶液、触手培养舱等，用于捕获、改造并持续榨取少年英雄的精液与能量。

魔神会制定了**「诸神黄昏」计划**，意图控制守护城市的少年英雄以夺取第三区控制权。魔神会内部规则：驯服少年英雄须在会内登记身份，为主教私人财产不可侵犯，拥有优质S级少年英雄的效忠被视为地位的象征。

## 1.6 特殊少年与神纹/淫纹机制

部分少年因先天或后天因素，成为"特殊容器"。他们的体液（尤其是精液）含有高纯度元素能量，可显著提升吸收者的元素亲和度与异能强度。这类少年往往拥有更高的战斗天赋与魔力亲和，同时也更容易吸引高级魔兽与魔神会的注意。

魔神会通过特定能力在少年身上烙印淫纹，这些淫纹可与少年自身神性融合，演化成全新的**神纹**。神纹一旦形成，将永久烙印于少年小腹、后颈等部位，具备以下核心作用：

- 提升敏感度（特定部位如足底、乳首、后穴、前列腺等）
- 诱发精液上瘾、无法满足的渴求、羞耻玩法沉迷等心理与生理特性
- 改造第二性征或第二性器，强化容器容量与精液产量、质量
- 形成主奴契约或精神羁绊，限制反抗能力，同时通过持续刺激实现高强度榨精
- 神纹可与英雄战衣、机械改造、御兽契约等结合，进一步放大效果

英雄战衣通常为连体紧身胶质或液态纳米材质，高度贴合身体，凸显肌肉线条、胸部弧度、腹肌、胯间鼓包与臀部轮廓。在神纹加持下，战衣可分泌辅助液体、释放微型触手，进一步刺激敏感部位，实现无时无刻的榨精与能量采集。

# 二、核心力量体系

## 2.1 六项战斗数值

- **力量、魔法、物防、魔防、体力、速度**
- 等级划分：C至SSS级
- 普通人普遍为B到A级，少数为S级

## 2.2 精液质量体系

- 能提升异能实力或让普通人觉醒
- 等级划分：B至SSS级
- A级及以上精液才对人类有效
- 年龄越小、身体机能越好的异能者，精液质量越高
- 特殊少年精液可显著增强元素亲和、提升异能，是魔神会争夺的核心资源
- 射出后几分钟内基因细胞失活分解为元素能量，无法用于基因改造

## 2.3 身体敏感度体系

- 普通人各项敏感度均为C或B（敏感部位为B）
- 特殊天赋或经身体改造可达A
- 极其稀少能达到S或以上
- SSS级为"第二性器"级别

## 2.4 第二性器

由后期针对弱点的身体开发改造而出现，敏感度极大提升。各角色第二性器各异：问天为足部SSS、子夜为肉棒SSS、千瑞为胸腹腿内侧SSS、空耀为后穴SSS、凯风为时空腺体SSS、嘉陵为分身共享感官SSS、阚泽为魔能系带SSS、星野为足部SSS。

## 2.5 堕落神装

**通用核心功能**：极致裸感·动态束紧·智能通道·能量收集

- 本质是神圣而具有强大超凡力量的装备，有自身的灵智

- 在神装基础上特殊改造加上宿主精血，形成智慧共生体
- 吸食宿主日常残留的精液维生
- 能够与宿主情感相同，洞悉宿主心之所念，适时刺激宿主身体
- 在宿主欲火难耐时会刺激宿主身体，带来强烈快感
- 在宿主败北后会根据精神状态进行相应反应使宿主快速陷入情欲
- 在战斗形态时刺激敏感部位暴露弱点；战损/战败形态更色情化

- **指腹系带揉压**：拇指指腹对准龟头腹侧系带做定点画圈揉压
- **马眼挑逗**：小指指腹轻点尿道口快速振动或指甲盖轻刮马眼边缘
