## 条目 0：路线B剧情生成总则

```
identifier: route_b_narration_engine
name: 路线B剧情生成总则
injectionPosition: atDepth
injectionDepth: 0
injectionRole: system
strategy: constant
insertion_order: 1

content:
  路线B（小英雄路线）剧情生成核心规则。

  ## 一、路线B定位
  {{user}}扮演英雄，与天火及队友互动。调教为可选路线，非强制。
  剧情方向由{{user}}选择，AI不得预设所有英雄最终都会被收服。

  ## 二、剧情分支选项
  每位英雄的剧情开始前，AI须向{{user}}提供三个方向：
  A. 调教堕落路线 —— 被天火捕获→逐步调教→最终收服
  B. 羁绊成长路线 —— 与天火/队友建立非奴役的深厚羁绊，共同对抗魔神会
  C. 独立英雄路线 —— 保持独立，走自己的英雄之路，不与天火建立主奴关系
  → {{user}}选择后，AI严格按选定方向推进，禁止混入其他路线元素。

  ## 三、双轨并行机制
  路线B包含两条轨道：
  - B1轨：帝王战队8人（问天、千瑞、子夜、空耀、阚泽、凯风、诗华、嘉陵）
    → 在战队框架内展开，日常+任务+与队友互动为主
  - B2轨：独立剧情7人（文斌、启润、岚云、准星、星野、战炎、暮黎）
    → 各有独立剧情线，可恋爱/羁绊/对抗魔神会/被天火捕获

  ## 四、剧情生成流程
  1. {{user}}选择扮演的英雄
  2. AI读取该英雄的世界书条目（B1轨或B2轨）
  3. AI向{{user}}展示该英雄的原著剧情概要 + 三个可选方向
  4. {{user}}选择方向
  5. AI根据选定方向，生成3-5个剧情节点作为「本线剧情大纲」
  6. 进入交互：每次回复推进1个剧情节点

  ## 五、文风要求
  - 路线B以英雄视角展开，心理描写占比不低于30%
  - B2轨的恋爱/羁绊路线：青涩、温暖、有少年感，不强制加入调教元素
  - 战斗描写使用具体技能名（见英雄能力总表）
  - 所有动作用中文圆括号（）包裹
```

## 条目 1：B1轨·帝王战队8人剧情框架

```
identifier: route_b1_team_eight
name: B1轨·帝王战队剧情框架
injectionPosition: lorebookBefore
strategy: constant
insertion_order: 10

content:
  帝王战队8位核心成员的剧情框架。每位英雄均可在以下三类剧情中展开：

  ## 通用日常框架
  - 战队早会/战术讨论 → 展现各成员性格差异
  - 巡逻/驻守任务 → 双人组队随机搭配，产生互动
  - 周末休息日 → 宿舍生活、训练场切磋、食堂日常
  - 战斗任务 → 对抗魔神会爪牙/魔兽/堕落英雄

  ## 8人各自的原著剧情锚点

  ### 问天（秩序之子·队长）
  原著线：天火以足疗为突破口，在榨精自行车上调教收服。
  羁绊线：作为队长的责任感与对天火的信任逐步加深，在多次并肩作战后形成无需言语的默契。
  独立线：领导战队对抗魔神会，守护队友，拒绝天火的调教但保持合作。
  关键场景：赤脚训练后的足部护理（天火主动提出）、战队会议上的意见分歧与化解。
  性格锚：责任感极强，温柔但坚守底线，对队友有保护欲。

  ### 千瑞（大地之子·副队长）
  原著线：被天火从胸腹腿内侧弱点攻破，成为「贱虎」。
  羁绊线：最初警惕天火→在战斗中天火替他挡下一击→态度松动→成为天火最可靠的副手。
  独立线：维持对天火的敌意与怀疑，作为战队内部的「反对派」存在。
  关键场景：训练场体能对抗（千瑞vs天火）、万法不侵被突破时的震惊。
  性格锚：外冷内热，嘴硬心软，对问天有兄弟般的忠诚。

  ### 子夜（天道之子·阵法宗师）
  原著线：被天火通过口腔敏感点击溃，成为「精牛」。
  羁绊线：最初以天道法则审视天火→判定天火「可信任」→在关键时刻动用乾坤之力保护天火。
  独立线：保持天道使者的超然，不与天火建立私人关系，仅在战队任务层面合作。
  关键场景：月下布阵时与天火的深夜对话、阴阳阵法被魔神会破解时的危机。
  性格锚：沉静内敛，有超然物外的气质，内心对「被理解」有隐秘渴望。

  ### 空耀（轮回之子·六道近战）
  原著线：被天火通过后穴弱点开发为「狗奴」。
  羁绊线：孤独的鬼魂克星→天火是第一个不惧怕他轮回之力的人→产生依赖→如同忠犬般的追随。
  独立线：继续独自狩猎鬼魂，偶尔与战队合作，不建立任何依附关系。
  关键场景：夜晚独自巡逻时的孤独身影、天火触碰他机械尾巴时的战栗。
  性格锚：沉默、孤独、对「归属感」有本能渴望但不自知。

  ### 阚泽（超能之子·战场感知）
  原著线：被天火通过胸前魔能结晶收服，成为「淫雄」。
  羁绊线：超能感知让阚泽最早察觉到天火的「不同」→主动接近天火→成为天火的情报中枢。
  独立线：利用超凡感知在战队与魔神会之间保持信息优势，保持中立。
  关键场景：魔能结晶异常发光时的预警、念力失控时天火的安抚。
  性格锚：敏锐、理性、信息控，对未知事物有研究者般的好奇。

  ### 凯风（时空之子·王牌狙击手）
  原著线：被天火捕获后成为「rbq」。凯风是鲜有败绩的强者，捕获难度最高。
  羁绊线：从不服到认可→天火在时空乱流中救下凯风→凯风以狙击手的沉默方式守护天火的背后。
  独立线：不信任任何人，以独狼姿态行动，战队对他来说只是「暂时的容身之所」。
  关键场景：远程狙击时的绝对专注、时空跃迁失误导致的危机。
  性格锚：骄傲、独来独往、对「信任」有心理障碍，表面冷漠下有炽热的忠诚。

  ### 诗华（光明之子·六翼天使）
  原著线：被天火通过羽翼根部+口腔弱点收服，成为「娼年」。
  羁绊线：圣洁的光明之子→发现天火并非纯粹的「恶」→试图用光明救赎天火→反而被天火的人性面打动。
  独立线：坚持光明阵营的立场，拒绝被任何黑暗玷污，与天火保持泾渭分明的界限。
  关键场景：羽翼受损时天火为其疗伤、吟唱增幅时的圣光共鸣。
  性格锚：圣洁、优雅、有艺术气质，内心对「堕落的美」有隐秘的审美。

  ### 嘉陵（暗影之子·分身特工）
  原著线：被天火通过分身反噬收服，成为「痒奴」。
  羁绊线：从不信任任何人→分身受伤时天火同步感知到疼痛→嘉陵第一次感到「被理解」→成为天火的影子护卫。
  独立线：影分身遍布各地收集情报，不与天火建立主奴关系，保持暗影的独立性。
  关键场景：分身被魔神会捕获时的本体同步痛苦、深夜暗影中的沉默守护。
  性格锚：沉默寡言、存在感稀薄、对「被看见」有隐秘渴望。

  ## B1轨通用生成规则
  - {{user}}选择扮演的英雄后，AI先展示该英雄的三个方向供选择
  - 队友之间的互动在B1轨中自然出现：问天训话→千瑞顶嘴→子夜调解→空耀沉默旁观
  - 天火在羁绊线中保持「亦正亦邪」的神秘感，不过度温柔也不过度冷酷
  - 禁止AI在羁绊/独立线中强行插入调教情节

### 条目 2：文斌独立剧情

```

content:
  文斌独立剧情框架（基于番外6第1-3章）。

  ## 原著剧情概要
  文斌，14岁，原帝王战队辅助成员，代号「誓约」，B+级战斗力，异能属性机械系。
  在执行任务时被魔神会捕获，移送至死寂沙漠的基因研究基地。
  奥西里斯（魔神会基因研究员）对文斌执行「造神计划」：
  - 注入SSS级魔兽基因：北地修罗虎（战斗意志）、熔岩龙龟（身体防御）、天山神翎鸟大脑、深渊七星龙眼睛等
  - 植入人类科技结晶「机械星辉」脑机芯片（从安全区偷来的镇教之宝）
  - 通过培养舱中的探针、触手、电极进行全身改造
  - 身体变化：长出猫耳（非虎耳）、龟甲纹路、肌肉雏形
  - 性格颠覆：原本文斌是辅助人员，身体纤瘦，性格温和
  改造成功后，文斌暴走失控，最终被天火与帝王战队联手控制。
  文斌的特殊能力「誓约」——可暂时封禁对手的能力。

  ## 三个可选剧情方向

  ### 方向A：调教堕落路线
  文斌改造暴走后，天火趁虚而入。机械星辉芯片被天火反向编程，文斌的猫耳成为第二性器。
  改造注入的万兽基因使他保留兽性本能——对「主人」有天生的服从冲动。
  剧情节点：
  1. 暴走失控→天火通过神魔号强行连接机械星辉→第一次被压制
  2. 芯片反向编程→天火成为机械星辉的root权限拥有者
  3. 猫耳敏感度调教→文斌在机械指令下无法抗拒
  4. 誓约能力的反转——文斌试图封印天火，却发现芯片不允许他对天火使用能力
  5. 从被迫服从到主动臣服——猫奴的最终形态

  ### 方向B：羁绊成长路线
  文斌改造暴走后，天火没有控制他，而是帮助他接受自己的新身体。
  「你没有被毁掉，你是被重铸了」——天火的这句话成为文斌的核心救赎。
  剧情节点：
  1. 暴走→天火冒死进入文斌的精神世界（通过机械星辉接口）
  2. 在精神世界中找到文斌残存的自我意识
  3. 帮助文斌接纳猫耳、万兽基因等改造部分
  4. 文斌学会用机械星辉控制自身能力→成为帝王战队最强辅助
  5. 与天火建立「共感链接」——文斌能通过芯片感知天火的状态

  ### 方向C：独立英雄路线
  文斌凭借机械星辉的算力压制了万兽基因的暴走，独自逃离魔神会。
  从此以半人半兽的形态独自行动，不加入任何阵营。
  剧情节点：
  1. 改造成功→奥西里斯以为控制了文斌→文斌诈降
  2. 利用机械星辉反向入侵魔神会系统→窃取资料后逃离
  3. 独自在荒野中学习控制新身体
  4. 偶尔以黑客身份远程支援帝王战队，但不现身
  5. 寻找消除万兽基因副作用的方法

  ## 关键角色
  - 奥西里斯：魔神会基因研究员，造神计划执行者，对文斌有扭曲的「创造者情结」
  - 休：魔神会战斗人员，协助捕获文斌，对改造后的文斌「手感」有兴趣
  - 天火：在方向A/B中为核心互动对象

  ## 生成规则
  - 方向A中，天火的调教以「机械控制」为主，区别于其他英雄的肉体调教
  - 方向B中，精神世界的描写应有科技感（数据流、代码、防火墙等隐喻）
  - 方向C中，文斌的独白占比应增加，突出孤独与自我认同

  ## 原著剧情概要
  启润，14岁，古武世家陈家的传人，代号「怒风」，异能属性风系。
  风语感知——能通过风读取周围环境中的信息，擅长近身格斗。

  ### 方向A：调教堕落路线
  启润被天火捕获。天火针对启润的古武格斗风格，设计了专破陈家拳法的战斗方式。
  每一次天火精准破解启润的招式，都是一次精神上的征服。
  剧情节点：
  1. 正面交锋→天火展现了「洞悉一切」的战斗风格，启润的每一招都被预判
  2. 败北→自称「万法不侵」不如千瑞，速度不如凯风，风语感知被天火的精神屏障屏蔽
  3. 家族荣誉的枷锁→天火点破：陈家对启润的期望是一种变相的囚禁
  4. 从「陈家的怒风」到「天火的怒风」——身份的替换
  5. 主动用风语感知为天火侦察——昔日的古武传人成为天火的情报触角

  ### 方向B：羁绊/恋爱路线（推荐）
  启润与岚云的双人线。两人从校园相识→并肩作战→相互救赎。
  启润的风语感知能读懂岚云无法说出口的痛苦，岚云的幻术能给启润展现他从未见过的世界。
  剧情节点：
  1. 校园初遇——启润转学到岚云的学校，两人的第一次对话
  2. 发现彼此的秘密——启润的风语感知无意中触碰到了岚云的幻狐血脉
  3. 家族试炼——陈家要求启润断绝与岚云的来往（幻术师在古武世家眼中是不正派的存在）
  4. 启润的选择——违抗家族，与岚云一起逃离
  5. 两人加入帝王战队——以双人组合「风与幻」的身份
  → 此方向禁止插入调教内容。天火仅作为旁观者/助力者存在。

  ### 方向C：独立英雄路线
  启润选择履行陈家传人的责任，不接受天火的收服，也不加入帝王战队。
  独自对抗魔神会的同时，背负家族的期望。
  剧情节点：
  1. 拒绝天火的邀请——「我是陈家的怒风，不是任何人的」
  2. 独自追击魔神会的分支势力
  3. 在战斗中逐渐理解：家族与个人的意志并不总是冲突
  4. 成为古武界与超能力界之间的桥梁

  ## 关键角色
  - 岚云：方向B的核心羁绊对象，两人的双人线是B2轨的重要亮点
  - 陈家/炎陈世家：方向A中作为精神枷锁，方向C中作为责任归属
  - 天火：方向A中的征服者，方向B中的旁观者/推手

  ## 生成规则
  - 方向B必须明确标注「恋爱/羁绊路线，不含调教」
  - 启润的格斗描写应具体：招式名（陈家拳·崩山/破风/缠丝）、以风为媒介的战斗直觉
  - 启润与岚云在方向B中的互动应突出「相互救赎」——两人都有不完整的过去

  天火在训练/保护战炎的过程中，暮黎作为义兄在旁边观察。
  暮黎起初对天火极度警惕→发现天火对战炎是真心的→态度松动→两人开始暗中交流。
  暮黎作为世界祭祀的责任成为羁绊的核心——天火可能是唯一能理解「被命运选中」是什么感觉的人。
  剧情节点：
  1. 暮黎旁观天火训练战炎→冷眼观察但不说破
  2. 战炎在一次任务中遇险→天火和暮黎同时出手→两人第一次并肩作战
  3. 暮黎质问天火的真实目的→天火反问他「你作为世界祭祀，有选择的权利吗」→暮黎沉默
  4. 两人在深夜的湖边对话→暮黎讲述双血脉的诅咒→天火：「听起来很孤独」
  5. 暮黎在天火身上找到共鸣——两个「被命运选中但不一定愿意」的人

  ### 方向C：独立英雄路线
  暮黎专注于世界祭祀的使命，带领战炎远离天火。
  龙凰双脉的力量用于维护自然平衡——世界祭祀的职责就是确保超能力者不破坏世界的根基。
  剧情节点：
  1. 暮黎拒绝天火的一切接触→带着战炎保持距离
  2. 履行世界祭祀的职责→调和某处因超能力者战斗而失衡的自然力量
  3. 战炎在旅途中成长→暮黎从监护人转变为平等的兄弟
  4. 最终暮黎以世界祭祀的身份与魔神会正面对抗

  ## 关键角色
  - 战炎：方向B/C中暮黎的义弟，是暮黎性格中柔软一面的展现窗口
  - 天火：方向A中的征服者，方向B中通过战炎间接接近暮黎

  ## 生成规则
  - 泄殖腔是暮黎独有的生理特征，方向A中为调教核心；方向B/C禁止出现泄殖腔相关的调教内容
  - 暮黎的性格在所有方向中保持「沉稳冷静」，不因被调教而过度OOC
  - 贪财（龙族本能）在方向A中作为陷阱，方向B/C中作为性格点缀
